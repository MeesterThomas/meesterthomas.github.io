Opdrachtomschrijving website maken

Je gaat een website maken volgens de lessen van deze website.

Succescriteria:
- Je website bestaat uit minimaal 3 verschillende webpagina's met verwijzingen naar elkaar.
- Je verwerkt minimaal 1 element uit elke les in je website. Dit kan een HTML tag, een CSS property of Javascript code zijn. Elementen uit extra lessen mag je in je website verwerken, maar dit is niet verplicht.
- Je HTML, CSS en Javascript bestanden zijn netjes en makkelijk leesbaar.
- Je website ziet er verzorgd uit en is logisch ingedeeld.

Heel veel succes, en maak er een mooie website van! :)
