Extra les 6: Escape Room


Escape Room

In deze les leer je via een stappenplan hoe je een escape room kan maken met HTML, CSS en Javascript. Ditzelfde stappenplan wordt ook gebruikt bij het maken van andere games.

Stap 1: Game Design

We beginnen waar elke game begint: een idee. Het is belangrijk dat je een idee hebt hoe je game eruit komt te zien. Wat kan je allemaal doen in je game? Dit noemen we ook wel Game Design.

Bij een escape room heb je veel vrijheid om zelf te bedenken hoe je hem in gaat vullen. Wat voor elementen wil je gebruiken? Denk daarover na. In de escape room die wij gaan maken moet de speler drie vragen beantwoorden. De antwoorden kan je op verschillende manieren vinden:
- Er is een stuk verborgen tekst. Deze tekst moet je selecteren om het te kunnen lezen.
- Er staat een antwoord op een andere webpagina. Je moet hiervoor op een linkje drukken.
- Een van de antwoorden staat tijdelijk in de invulbalk van een andere vraag.

Natuurlijk kan je zelf andere elementen kiezen, of een combinatie maken van. Zo wordt het helemaal jouw eigen escape room!

Tip: Liever goed gejat dan slecht bedacht. Kijk ook eens naar andere escape rooms. Hoe zien die eruit? Wat vind je er goed aan, en wat niet?

Stap 2: HTML

Nu je weet hoe je game eruit komt te zien en wat je erin wilt hebben, is het tijd om de HTML tags toe te voegen aan je webpagina. Kijk hierbij goed naar wat je nodig hebt.

Voor je escape room moet je goed kijken naar de elementen die je wilt toevoegen. Het is handig om je escape room in meerdere fases in te delen. Bijvoorbeeld, eerst moet je op start drukken (fase 1), daarna moet je 4 vragen beantwoorden (fase 2), daarna moet je nog een laatste vraag beantwoorden (fase 3) en tot slot krijg je een scherm dat je hebt gewonnen (fase 4).

Als je deze verschillende fases hebt, kan je ze makkelijk in aparte div elementen plaatsen (zoals hiernaast). De div elementen kan je dan verbergen en weer laten zien wanneer jij dat wilt.

Voor de escape room die ik ga maken heb je de elementen nodig die hiernaast staan.

Op regel 10 staat een input element. Hier kan je dus tekst invoeren. Hier wil ik dat tijdelijk een antwoord staat. Daarvoor gebruik ik het stukje placeholder="". Tussen de haakjes komt dan de tekst te staan. 


    <body>
        <div id="verborgen0">
            <button id="startKnopER" onclick="StartER()">Start</button>
        </div>
        <div id="verborgen1">
            <input type="text" id="antwoord0">
            <button id="check0" onclick="ERKnop(0)">Check</button>
            <br><input type="text" id="antwoord1">
            <button id="check1" onclick="ERKnop(1)">Check</button>
            <br><input type="text" id="antwoord2" placeholder="Het tweede antwoord is 2.">
            <button id="check2" onclick="ERKnop(2)">Check</button>
            <p class="verborgenTekst">Het eerste antwoord is 3.</p>
            <a href="6extra.html" target="_blank">Info!</a>
        </div>
        <div id="verborgen2">
            <p>Het is gelukt!</p>
        </div>
    </body>

Stap 3: De noodzakelijke opmaak (CSS)

Je hebt het HTML bestand nu voor een groot deel klaar. Nu is het belangrijk om de noodzakelijke opmaak in orde te maken.

Met de noodzakelijke opmaak wordt de bedoeld dat je alleen moet kijken wat er echt moet veranderen om je game werkende te krijgen. Kleuren en mooi maken kan altijd nog.

Ik wil dat de div met id verborgen1 en verborgen2 nog even niet te zien zijn. Daarvoor gebruik ik display: none; De verborgentekst wordt verborgen als color en background-color dezelfde kleur hebben. 


#verborgen1
{
    display: none;
}
#verborgen2
{
    display: none;
}
.verborgenTekst
{
    color: rgb(11, 11, 177);
    background-color: rgb(11, 11, 177);
}

Stap 4: Javascript

Nu je alle elementen op de juiste plaats hebt, is het tijd voor het programmeren van je game met javascript.

Het is belangrijk om stap voor stap op te bouwen. Als je naar je webpagina gaat met je escape room, zie je alleen maar een startknop die niks doet als je erop drukt. Laten we daar verandering in brengen!

Maak een function in je javascript bestand die ervoor zorgt dat de goede elementen zichtbaar worden. Gebruik hiervoor display. 


function StartER()
{
    document.getElementById("verborgen0").style.display = "none";
    document.getElementById("verborgen1").style.display = "block";
}

Stap 4: Javascript

De vragen en hints zijn al zichtbaar, dus we moeten er alleen maar voor zorgen dat de check knoppen hun werk doen. Daarvoor maken we een nieuwe function aan.

We gaan controleren of de antwoorden kloppen. Alleen als alle 3 de antwoorden goed zijn, gaat de escape room door naar de volgende fase. Dit moeten we dus wel ergens bijhouden. Dit doen we met de variabele goed. Dit wordt een array met 3 onderdelen. Voor nu zijn ze nog allemaal false (fout).

In onze nieuwe function checken we per vraag of het antwoord wat is opgegeven klopt. Als dit zo is, veranderen we de waarde uit de array goed van false (fout) naar true (juist). Ook veranderen we de achtergrondkleur zodat de speler weet dat het antwoord goed was.

Dit herhalen we voor de andere vragen. 


let goed = [false, false, false];

function StartER()
{
    document.getElementById("verborgen0").style.display = "none";
    document.getElementById("verborgen1").style.display = "block";
}

function ERKnop(knopNummer)
{
    if (knopNummer == 0 && document.getElementById("antwoord0").value == "3")
    {
        goed[knopNummer] = true;
        document.getElementById("antwoord0").style.backgroundColor = "rgb(132, 132, 215)";
    }
    else if (knopNummer == 1 && document.getElementById("antwoord1").value == "2")
    {
        goed[knopNummer] = true;
        document.getElementById("antwoord1").style.backgroundColor = "rgb(132, 132, 215)";
    }
    else if (knopNummer == 2 && document.getElementById("antwoord2").value == "1")
    {
        goed[knopNummer] = true;
        document.getElementById("antwoord2").style.backgroundColor = "rgb(132, 132, 215)";
    }
}

Stap 4: Javascript

We zijn er al bijna! We moeten nog 2 dingen toevoegen.

Als eerste checken we aan het begin van de function of het antwoord hiervoor al goed was. Als dit zo is, dan hoeft er niks te gebeuren namelijk. Dan gebruiken we return.

Als tweede moeten we aan het einde van de function checken of alle antwoorden juist zijn. Als dit zo is, dan kunnen we door naar de volgende fase. Hierbij veranderen we dus weer welke elementen zichtbaar zijn. Dit doen we op dezelfde manier als bij de StartER() function. 


let goed = [false, false, false];

function StartER()
{
    document.getElementById("verborgen0").style.display = "none";
    document.getElementById("verborgen1").style.display = "block";
}

function ERKnop(knopNummer)
{
    if (goed[knopNummer == true])
        return;
    if (knopNummer == 0 && document.getElementById("antwoord0").value == "3")
    {
        goed[knopNummer] = true;
        document.getElementById("antwoord0").style.backgroundColor = "rgb(132, 132, 215)";
    }
    else if (knopNummer == 1 && document.getElementById("antwoord1").value == "2")
    {
        goed[knopNummer] = true;
        document.getElementById("antwoord1").style.backgroundColor = "rgb(132, 132, 215)";
    }
    else if (knopNummer == 2 && document.getElementById("antwoord2").value == "1")
    {
        goed[knopNummer] = true;
        document.getElementById("antwoord2").style.backgroundColor = "rgb(132, 132, 215)";
    }
    if (goed[0] == true && goed[1] == true && goed[2] == true)
    {
        document.getElementById("verborgen1").style.display = "none";
        document.getElementById("verborgen2").style.display = "block";
    }
}

Stappen 5 en 6: Testen en aanpassen

Je game is nu zo goed als af! Nu komt de laatste fase: testen en aanpassen. In deze fase ga je je game spelen en aanpassingen maken die je nodig vindt.
