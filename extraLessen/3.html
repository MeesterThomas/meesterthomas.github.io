Extra les 3: Navigatiebalk


Navigatiebalk

In deze les leer je hoe je een navigatiebalk kan maken voor jouw website. Een navigatiebalk kan erg handig zijn om de bezoekers van je website naar verschillende webpagina's te sturen.

Het is dus wel belangrijk dat je al verschillende webpagina's hebt, anders wordt het lastig om een navigatiebalk te maken!

Stap 1: Een balk aanmaken

We beginnen met het aanmaken van een balk waar de elementen in komen te staan. Dit doen we in een div tag in een HTML bestand. Deze div kan je vervolgens overal op je webpagina plaatsen. We geven deze div een class. Ik geef hem de classnaam "navBalk".

Nu we de navBalk in ons HTML bestand hebben, gaan we hem opmaken in CSS. Als eerste wil ik dat de balk de hele breedte van de webpagina inneemt. Dat doe ik met "width: 100%". Ook geef ik hem een achtergrondkleur. Dit wordt voor mij zwart. 


<div class="navBalk">
</div>

-------------------------------
.navBalk
{
    width: 100%;
    background-color: black;
}

Stap 2: De balk vullen

Nu is het tijd om de balk te vullen met elementen. Dit kan tekst zijn, of dit kunnen afbeeldingen zijn. Ik raad niet aan om buttons te gebruiken.

Voor mijn voorbeeld gebruik ik tekst, maar het werkt ook met afbeeldingen (gebruik daarvoor de img tag). Elk element van de balk krijgt van mij 1 regel. 


<div class="navBalk">
    <p>Home</p>
    <p>Codeboek</p>
    <p>Lessen</p>
    <p>Extra Lessen</p>
    <p>Games</p>
</div>

-------------------------------
.navBalk
{
    width: 100%;
    background-color: black;
}

Stap 3: De elementen opmaken

Je hebt je elementen nu aangemaakt, nu is het tijd om ze mooi te maken. Hoe je dit doet is natuurlijk aan jezelf, maar ik omschrijf welke stappen ik neem. Het handigst is om een nieuwe class aan te maken voor de elementen. Dit kan dezelfde class zijn. Mijn naam wordt "navBalkElement".

Mijn achtergrond is zwart, dus wil ik mijn tekst wit maken. Dit doe ik met color.

Ik wil mijn elementen naast elkaar hebben, niet onder elkaar. Daarvoor verander ik de "display" property van de balk naar "flex". Dit zorgt ervoor dat alle elementen naast elkaar gepropt worden. Om het netjes te verdelen, gebruik ik de "width" property bij de elementen zodat ze elk 20% van de balk innemen.

Ik wil mijn tekst in het midden van elk element. Daarvoor gebruik ik de property "text-align" bij de elementen van de balk.

Mijn tekst mag nog iets groter worden. Daarvoor gebruik ik de property "font-size".

Tot slot wil ik een rand rondom de elementen. Dit wil ik in de kleur blauw. Dit doe ik met de "border" properties. 


<div class="navBalk">
    <p class="navBalkElement">Home</p>
    <p class="navBalkElement">Codeboek</p>
    <p class="navBalkElement">Lessen</p>
    <p class="navBalkElement">Extra Lessen</p>
    <p class="navBalkElement">Games</p>
</div>

-------------------------------
.navBalk
{
    width: 100%;
    background-color: black;
    color: white;
    font-size: 32px;
    display: flex;
}
.navBalkElement
{
    width: 20%;
    text-align: center;
    border-width: 3px;
    border-style: solid;
    border-color: blue;
}

Stap 4: De linkjes maken

Je navigatiebalk ziet er nu helemaal goed uit, maar hij werkt nog niet. Daar gaan we verandering in brengen!

Met een a tag kan je een link maken naar een andere webpagina. Dat doen we door de a tag om het element heen te zetten. Zie het voorbeeld hiernaast om te kijken wat ik ermee bedoel.

Als we dit doen, kan het zijn dat de opmaak verandert. Om dit te voorkomen, kan je de class toekennen aan de a tag in plaats van de p tag. Als het goed is moet de opmaak verder gewoon blijven kloppen.

Nu ziet mijn opmaak er weer bijna hetzelfde uit, alleen is mijn tekst nu blauw geworden met een streep eronder. Dat komt omdat het een linkje is geworden. Ook weet een gebruiker niet snel of je op de knop kan drukken, aangezien hij niet verandert als je eroverheen beweegt met je muis.

Dit kunnen we allemaal regelen in CSS. Binnen de class "navBalkElement" zet ik de volgende 2 dingen. Met text-decoration kunnen we de streep verwijderen, en met color kunnen we de kleur van de tekst aanpassen.

Tot slot wil ik dus dat de knop verandert van kleur als ik eroverheen beweeg met mijn muis. Hiervoor heb ik een nieuw kopje nodig bij mijn CSS. Een kopje wat alleen activeert als ik eroverheen beweeg. Gelukkig kunnen we dat doen door ":hover" achter een class (of tag of id) te zetten. Bij dit kopje verander ik de "background-color". 


<div class="navBalk">
    <a href="https://meesterthomas.github.io/index.html" class="navBalkElement"><p>Home</p></a>
    <a href="https://meesterthomas.github.io/codeboek.html" class="navBalkElement"><p>Codeboek</p></a>
    <a href="https://meesterthomas.github.io/lessen.html" class="navBalkElement"><p>Lessen</p></a>
    <a href="https://meesterthomas.github.io/extralessen.html" class="navBalkElement"><p>Extra Lessen</p></a>
    <a href="https://meesterthomas.github.io/games.html" class="navBalkElement"><p>Games</p></a>
</div>

-------------------------------
.navBalk
{
    width: 100%;
    background-color: black;
    color: white;
    font-size: 32px;
    display: flex;
}
.navBalkElement
{
    width: 20%;
    text-align: center;
    border-width: 3px;
    border-style: solid;
    border-color: blue;
    text-decoration: none;
    color: white;
}
.navBalkElement:hover
{
    background-color: grey;
}

Stap 5 (Extra): De balk in beeld houden

Nu blijft je navigatiebalk op een plaats op de webpagina staan. Als je een lange webpagina hebt, kan het handig zijn als de navigatiebalk mee scrollt.

Dat kunnen we doen door de "position" property van de navBalk class te veranderen naar "sticky". Dit zorgt ervoor dat hij meescrollt. Om dit voor elkaar te krijgen, moeten we wel benoemen waar op het scherm de navigatiebalk moet komen. Dit doe je met de properties "top", "bottom", "left" en "right". Ik wil hem bovenaan, dus zet ik mijn "top" naar 0. 


<div class="navBalk">
    <a href="https://meesterthomas.github.io/index.html" class="navBalkElement"><p>Home</p></a>
    <a href="https://meesterthomas.github.io/codeboek.html" class="navBalkElement"><p>Codeboek</p></a>
    <a href="https://meesterthomas.github.io/lessen.html" class="navBalkElement"><p>Lessen</p></a>
    <a href="https://meesterthomas.github.io/extralessen.html" class="navBalkElement"><p>Extra Lessen</p></a>
    <a href="https://meesterthomas.github.io/games.html" class="navBalkElement"><p>Games</p></a>
</div>

-------------------------------
.navBalk
{
    width: 100%;
    background-color: black;
    color: white;
    font-size: 32px;
    display: flex;
    position: sticky;
    top: 0px;
}
.navBalkElement
{
    width: 20%;
    text-align: center;
    border-width: 3px;
    border-style: solid;
    border-color: blue;
    text-decoration: none;
    color: white;
}
.navBalkElement:hover
{
    background-color: grey;
}
