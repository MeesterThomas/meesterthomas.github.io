Extra les 2: Inspecteren


Inspecteren

Als je naar een willekeurige webpagina gaat en je klikt met de rechtermuisknop, krijg je een aantal opties. Helemaal onderaan komt inspecteren te staan. Als je hierop klikt, gaat er een wereld voor je open, en daar wil ik je graag in meenemen!

Let op: Dit kan je niet doen op de Chromebook van school, aangezien deze beveiligd is voor deze functie. Dit moet je dus thuis proberen op een andere laptop of computer.

Elements

Nadat je op inspecteren hebt geklikt, zie je een heleboel verschillende termen. Een paar daarvan ga ik hier toelichten. De eerste is het kopje "elements" wat vanboven staat. Hier zie je het hele HTML bestand van de website (mogelijk ingeklapt).

Als je dit HTML bestand bekijkt, kan je met je muis over de code heen zweven. Als je dit doet, merk je dat delen van de webpagina gekleurd worden. Het gekleurde deel op de webpagina is het deel wat overeenkomt met de code waar je overheen beweegt. Zo kan je dus bekijken waaruit dat deel van de webpagina is opgebouwd!

Dit "elements" kopje kan je ook aanpassen. Als je dubbel klikt op de code, kan je het aanpassen. Dat betekent dat je de HTML van de website aan kan passen, waaronder de tekst!

Het aanpassen van de tekst doe je op deze manier alleen op je eigen computer. Dat betekent dat jij misschien een leuke tekst hebt staan, maar dat kan iemand anders die toevallig ook op die website zit niet lezen.

Styles

Onder "elements" zie je waarschijnlijk "styles". Als je een HTML element aanklikt bij "elements", kan je bij "styles" zien welke CSS properties er op dit element werken.

Ook hier kan je weer aanpassingen maken. Je kan CSS properties veranderen, of uitschakelen. Probeer het eens uit! Ook hierbij geldt dat alleen jij deze wijzigingen kan zien.

Opdracht: Ga los!

Kies een webpagina, en probeer hem maar eens helemaal aan te passen! Verander de tekst, kies andere kleuren, verwijs naar andere afbeeldingen, ga los!

Console

Helemaal vanonder zie je een kopje met "console". In de console komen alle errors of meldingen te staan van jouw website. Deze komen er alleen te staan als je webpagina Javascript gebruikt.

Als je je Javascript code aan het testen bent, kan de console heel erg handig zijn. Je kan hier bijvoorbeeld zien wat er fout gaat bij jouw code.

Sources

Als je terug gaat naar het kopje "elements" vanboven, zie je daar in dat balkje wat andere termen. Als je op "sources" klikt, dan ga je naar de map waar de website in staat. Je ziet hier dus alle bestanden die de webpagina nodig heeft!

Als je op een bestand klikt, bijvoorbeeld een Javascript bestand, kan je ze hier ook aanpassen. Vergeet niet op ctrl + s te drukken om de wijzigingen op te slaan... voor jou natuurlijk. Anderen krijgen dit wederom niet te zien.

Paginabron weergeven

Ik heb nog een extra voor je! Als je op een willekeurige webpagina klikt met de rechtermuisknop, zie je boven de optie inspecteren de optie "paginabron weergeven". Als je hierop klikt, krijg je het hele HTML bestand van deze webpagina te zien!
