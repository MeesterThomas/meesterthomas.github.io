Extra les 4: Quiz


Quiz

In deze les leer je via een stappenplan hoe je een quiz kan maken met HTML, CSS en Javascript. Ditzelfde stappenplan wordt ook gebruikt bij het maken van andere games.

Stap 1: Game Design

We beginnen waar elke game begint: een idee. Het is belangrijk dat je een idee hebt hoe je game eruit komt te zien. Wat kan je allemaal doen in je game? Dit noemen we ook wel Game Design.

Voor de quiz begin je vooral met nadenken over de quiz zelf. Probeer de onderstaande vragen voor jezelf te beantwoorden:
- Welke vragen ga je stellen of welke soort vragen ga je stellen?
- Wat voor soort antwoord kan de speler geven, open vragen, gesloten vragen of allebei?
- Houd je een score bij? Zo ja, hoe doe je dat? Doe je dat op het scherm (dat de speler het kan zien), of geef je de resultaten aan het eind?
- Welke andere functies en elementen zou je toe willen voegen?

Tip: Liever goed gejat dan slecht bedacht. Kijk ook eens naar andere (online) quizzen. Hoe zien die eruit? Wat vind je er goed aan, en wat niet?

Stap 2: HTML

Nu je weet hoe je game eruit komt te zien en wat je erin wilt hebben, is het tijd om de HTML tags toe te voegen aan je webpagina. Kijk hierbij goed naar wat je nodig hebt.

Voor de quiz moet je weer teruggrijpen op je game design. Je zal en tekst (p) tag nodig hebben om de vraag in te stellen. Vervolgens ook een vlak waarin je antwoord kan geven. Het is niet erg dat het er nog niet uitziet als een game, dat komt later.

Denk aan de volgende elementen:
- Voor het begin van je quiz: Een startknop die de quiz laat beginnen.
- Tekst om je vragen in te vermelden.
- Voor open vragen: Een input vlak waar je tekst in kan vullen en een knop om het antwoord in te leveren.
- Voor gesloten vragen: Knoppen met tekst voor de gesloten vragen (denk na hoeveel mogelijkheden je wilt).
- Voor een zichtbare score: Een tekstelement waar de score in kan staan. 


<!DOCTYPE html>
<html lang="en">
    <head>
        <title>Quiz</title>
        <meta charset="utf-8">
        <link rel="stylesheet" href="css/quiz.css">
        <script src="js/quiz.js"></script>
    </head>
    <body>
        <button>Start</button>
        <p>Vraagtekst</p>
        <button>Optie 1</button>
        <button>Optie 2</button>
        <button>Optie 3</button>
        <button>Optie 4</button>
        <p>Score: 0</p>
    </body>
</html>

Stap 3: De noodzakelijke opmaak (CSS)

Je hebt het HTML bestand nu voor een groot deel klaar. Nu is het belangrijk om de noodzakelijke opmaak in orde te maken.

Met de noodzakelijke opmaak wordt de bedoeld dat je alleen moet kijken wat er echt moet veranderen om je game werkende te krijgen. Kleuren en mooi maken kan altijd nog.

Een voorbeeld voor de quiz is elementen die als eerst niet zichtbaar zijn, onzichtbaar maken. Je begint met een startknop, maar de vraagtekst, opties en score zijn nu nog niet zichtbaar. Gebruik hiervoor display: none in CSS. 


.optieKnop
{
    display: none;
}
#vraagTekst
{
    display: none;
}
#scoreTekst
{
    display: none;
}

Stap 4: Javascript

Nu je alle elementen op de juiste plaats hebt, is het tijd voor het programmeren van je game met javascript.

Het is belangrijk om stap voor stap op te bouwen. Als je naar je webpagina gaat met je quiz, zie je alleen maar een startknop die niks doet als je erop drukt. Laten we daar verandering in brengen!

Maak een function in je javascript bestand die ervoor zorgt dat de goede elementen zichtbaar worden. Gebruik hiervoor display. 


function Start()

}

Stap 4: Javascript

Nu is het tijd om de goede vragen op te stellen voor de quiz. Dit doe je in een const variabele, want de vragen veranderen niet als je de code uitvoert.

Om het makkelijk te maken, maak je een array aan van alle vragen, waarin een array zit per vraag. In de array per vraag kan je verschillende punten kwijt. Bijvoorbeeld een vraagnummer, de vraag zelf, het goede antwoord en de foute antwoorden. Gebruik het voorbeeld hiernaast.

In het voorbeeld hiernaast zie je op elke regel een andere vraag. Binnen zo'n array, zie je verschillende variabelen tussen "", gescheiden met een ,. Deze staan voor variabelen.
- De eerste waarde is het nummer van de vraag. Zorg ervoor dat elke vraag een ander nummer heeft.
- De tweede waarde is de vraag.
- De derde waarde is het goede antwoord.
- De vierde, vijfde en zesde waardes zijn de andere antwoordopties. Deze zijn dus fout.

Maak je vragenlijst maar eens aan. Probeer minimaal 2 vragen te bedenken. Je kan later altijd nog terugkomen om vragen toe te voegen. 


const vragen = [

];

Stap 4: Javascript

Nu we de vragen hebben, is het tijd om die vragen te gaan stellen. Hiervoor maken we een nieuwe function aan het einde van de start function.

In deze function gaan we een vraag kiezen om te stellen. Het is handig om deze vraag te onthouden in de code, zodat we later het antwoord kunnen controleren. Daarvoor gebruiken we een array variabele (in het voorbeeld huidigeVraag).

We beginnen door huidigeVraag leeg te maken. Daarna gaan we kijken naar alle vragen die we hebben opgesteld. Dit doen we met een forEach loop. In deze loop laten we de eerste vraag opnemen in huidigeVraag. Vervolgens laten we de vraag zien in het tekstelement met id "vraagTekst". Dit doen we ook met de antwoordopties. 


const vragen = [
    ["0", "1+1=", "2", "3", "4", "1"],
    ["1", "2+2=", "4", "2", "5", "6"]
];
let huidigeVraag = [];

function Start()
{
    document.getElementById("startKnop").style.display = "none";
    document.getElementById("vraagTekst").style.display = "block";
    document.getElementById("optie0").style.display = "inline";
    document.getElementById("optie1").style.display = "inline";
    document.getElementById("optie2").style.display = "inline";
    document.getElementById("optie3").style.display = "inline";
    document.getElementById("scoreTekst").style.display = "block";
    StelVraag();
}
function StelVraag()
{
    huidigeVraag = [];
    vragen.forEach(vraag => {
        if (huidigeVraag.length == 0)
        {
            huidigeVraag = vraag;
        }
    });
    document.getElementById("vraagTekst").innerHTML = huidigeVraag[1];
    document.getElementById("optie0").innerHTML = huidigeVraag[2];
    document.getElementById("optie1").innerHTML = huidigeVraag[3];
    document.getElementById("optie2").innerHTML = huidigeVraag[4];
    document.getElementById("optie3").innerHTML = huidigeVraag[5];
}

Stap 4: Javascript

We kunnen de vragen en de antwoordopties nu laten zien. Tijd om ervoor te zorgen dat er wat gebeurd als je op een van de knoppen drukt. Hiervoor gebruiken we een nieuwe function.

Eerst gaan we checken of het antwoord goed is. Hierbij vergelijken we de tekst op de knop waar de gebruiker net op heeft gedrukt, met het juiste antwoord. Als dit overeen komt, dan is het antwoord goed!

Als het antwoord goed is, moet de score omhoog. Dat betekent dat we ook een score moeten hebben. Deze hebben we bij verschillende functions nodig, dus we benoemen de variabele bovenaan met let. In de start function, resetten we de score nog eens naar 0. Verder voegen we 10 toe aan de huidige score in de antwoord function als het antwoord goed is.

Ook willen we dat de volgende vraag gelijk weer komt. Daarom sluiten we de function af met StelVraag(). 


const vragen = [
    ["0", "1+1=", "2", "3", "4", "1"],
    ["1", "2+2=", "4", "2", "5", "6"]
];
let score = 0;
let huidigeVraag = [];

function Start()
{

    document.getElementById("optie2").style.display = "inline";
    document.getElementById("optie3").style.display = "inline";
    document.getElementById("scoreTekst").style.display = "block";
    score = 0;
    StelVraag();
}
function StelVraag()
{
    huidigeVraag = [];
    vragen.forEach(vraag => {
        if (huidigeVraag.length == 0)
        {
            huidigeVraag = vraag;
        }
    });
    document.getElementById("vraagTekst").innerHTML = huidigeVraag[1];
    document.getElementById("optie0").innerHTML = huidigeVraag[2];

    document.getElementById("optie2").innerHTML = huidigeVraag[4];
    document.getElementById("optie3").innerHTML = huidigeVraag[5];

function Antwoord(knopNummer)
{
    if (document.getElementById("optie" + knopNummer.toString()).innerHTML == huidigeVraag[2])

        score += 10;
    }

Stap 4: Javascript

Als je je quiz nu probeert, merk je misschien dat hij dezelfde vraag blijft stellen. We moeten ervoor zorgen dat hij checkt of een vraag al gesteld is of niet. Dit doen we met een variabele genaamd gesteldeVragen.

Bovenin het document noemen we deze variabele met let, en zeggen we dat hij leeg is. Hierin komen de nummers van de vragen die we gehad hebben. In de Start() function zeggen we ook nog een keer dat hij leeg is, om er zeker van te zijn.

In de StelVraag() function kijken we eerste of we net zoveel vragen gesteld hebben als er zijn. Als dit zo is, dan is de quiz klaar. Dan hoeven we dus geen nieuwe vragen meer te stellen. Dan gebruiken we return.

In de forEach loop controleren we nu extra of de vraag niet al is opgenomen in gesteldeVragen. Dan hoeven we hem namelijk niet meer te stellen. In dat if statement gebruiken we gesteldeVragen.push om ervoor te zorgen dat het nummer van de huidigevraag ook in gesteldeVragen komt te staan. 


const vragen = [
    ["0", "1+1=", "2", "3", "4", "1"],
    ["1", "2+2=", "4", "2", "5", "6"]
];
let gesteldeVragen = [];
let score = 0;
let huidigeVraag = [];

function Start()
{
    document.getElementById("startKnop").style.display = "none";
    document.getElementById("vraagTekst").style.display = "block";
    document.getElementById("optie0").style.display = "inline";
    document.getElementById("optie1").style.display = "inline";
    document.getElementById("optie2").style.display = "inline";
    document.getElementById("optie3").style.display = "inline";
    document.getElementById("scoreTekst").style.display = "block";
    gesteldeVragen = [];
    score = 0;
    StelVraag();
}
function StelVraag()
{
    if (gesteldeVragen.length >= vragen.length)
    {
        return;
    }
    huidigeVraag = [];
    vragen.forEach(vraag => {
        if (huidigeVraag.length == 0 && gesteldeVragen.includes(vraag[0]) == false)
        {
            gesteldeVragen.push(vraag[0]);
            huidigeVraag = vraag;
        }
    });
    document.getElementById("vraagTekst").innerHTML = huidigeVraag[1];
    document.getElementById("optie0").innerHTML = huidigeVraag[2];
    document.getElementById("optie1").innerHTML = huidigeVraag[3];
    document.getElementById("optie2").innerHTML = huidigeVraag[4];
    document.getElementById("optie3").innerHTML = huidigeVraag[5];
}
function Antwoord(knopNummer)
{
    if (document.getElementById("optie" + knopNummer.toString()).innerHTML == huidigeVraag[2])
    {
        score += 10;
    }
    StelVraag();
}

Stap 4: Javascript

We zijn al een heel eind! De quiz is wel wat makkelijk zeker? Dat komt omdat de goede antwoorden altijd als eerste staan. Laten we de antwoorden op een willekeurige plaats neerzetten.

Aan het einde van de StelVraag() function gaan we een variabele benoemen genaamd randomNummer. Dit wordt een getal van 0 tot 3 (dus 4 mogelijkheden).

Vervolgens benoemen we per mogelijkheid welke volgorde de antwoorden krijgen. De specifieke volgorde maakt niet veel uit, maar zorg ervoor dat het goede antwoord op alle 4 de knoppen terecht kan komen.

    {
        Einde();
        return;
    }
    huidigeVraag = [];
    vragen.forEach(vraag => {
        if (huidigeVraag.length == 0 && gesteldeVragen.includes(vraag[0]) == false)
        {
            gesteldeVragen.push(vraag[0]);
            huidigeVraag = vraag;
        }
    });
    document.getElementById("vraagTekst").innerHTML = huidigeVraag[1];
    let randomNummer = Math.floor(Math.random() * 4);
    if (randomNummer == 0)
    {
        document.getElementById("optie0").innerHTML = huidigeVraag[2];
        document.getElementById("optie1").innerHTML = huidigeVraag[3];
        document.getElementById("optie2").innerHTML = huidigeVraag[4];
        document.getElementById("optie3").innerHTML = huidigeVraag[5];
    }
    else if (randomNummer == 1)
    {
        document.getElementById("optie0").innerHTML = huidigeVraag[4];
        document.getElementById("optie1").innerHTML = huidigeVraag[5];
        document.getElementById("optie2").innerHTML = huidigeVraag[2];
        document.getElementById("optie3").innerHTML = huidigeVraag[3];
    }
    else if (randomNummer == 2)
    {
        document.getElementById("optie0").innerHTML = huidigeVraag[3];
        document.getElementById("optie1").innerHTML = huidigeVraag[2];
        document.getElementById("optie2").innerHTML = huidigeVraag[5];
        document.getElementById("optie3").innerHTML = huidigeVraag[4];
    }
    else if (randomNummer == 3)
    {
        document.getElementById("optie0").innerHTML = huidigeVraag[5];
        document.getElementById("optie1").innerHTML = huidigeVraag[4];
        document.getElementById("optie2").innerHTML = huidigeVraag[3];
        document.getElementById("optie3").innerHTML = huidigeVraag[2];
    }
}
