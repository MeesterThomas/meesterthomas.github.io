Extra les 1: Bestandspaden


Bestandspaden

In deze extra les krijg je extra uitleg over bestandspaden en hoe je ze kan gebruiken om naar een bestand te verwijzen.

Een bestandspad is een volgorde aan instructies voor de computer om een bestand te zoeken. Deze instructies beschrijf je op een bepaalde manier. De makkelijkste manier om een bestandspad te noteren, is om de instructie eerst zelf uit te voeren.

Stap 1: Zoek je huidige bestand op

Als eerst ga je in je website map je huidige bestand zoeken. Dit is het bestand waarin de verwijzing te staan komt. Deze zit voor mij in de map "extraLessen".

Voor mijn voorbeeld ga ik een afbeelding toevoegen op deze webpagina. Deze afbeelding staat in een andere map.

De HTML tag om een afbeelding te maken komt in les 3 aan bod. 


De opmaak van mijn mappen voor deze website:

Website (map)
    extraLessen (map)
        1.html (html bestand) Dit is het huidige HTML bestand
        2.html (html bestand)
    img (map)
        Lessen (map)
            Bestanden.png (png bestand) Dit is de afbeelding

Stap 2: Een map verlaten

Als je naar mijn mappenstructuur in het voorbeeld kijkt, zie je dat de afbeelding die ik wil gebruiken niet binnen de map "extraLessen" te vinden is. Dat betekent dat ik een stapje "omhoog" moet, naar de map "Website".

Om dit te doen, moet ik het volgende aan mijn bestandspad toevoegen: ../ 


<img src="../">

Stap 3: Een map openen

Ik zit nu dus in de map "Website". Ik hoef deze map niet te verlaten, want de afbeelding zit binnen deze map. Om naar de afbeelding toe te gaan, moet ik eerst een map openen, in dit geval de map "img".

Om dit te doen, moet ik de naam van de map aan het bestandspad toevoegen, gevolgd door /

In mijn voorbeeld, moet ik vervolgens nog een map openen. Dit doe ik op dezelfde manier. 


<img src="../img/Lessen/">

Stap 4: Het goede bestand kiezen

Ik zit nu in de map "Lessen". In deze map zit het bestand wat ik zoek. Nu ga ik het goede bestand kiezen. In mijn geval "Bestanden.png".

Om dit te doen, moet ik de naam van het bestand toevoegen, inclusief het bestandstype. Dit sluit je niet af met /

Nu heb je het juiste bestandspad! 


<img src="../img/Lessen/Bestanden.png">

De les zit erop!
