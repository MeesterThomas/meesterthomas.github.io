Extra les 5: Boter, kaas en eieren


Boter, kaas en eieren

In deze les leer je via een stappenplan hoe je een boter, kaas en eieren spel kan maken met HTML, CSS en Javascript. Ditzelfde stappenplan wordt ook gebruikt bij het maken van andere games.

Stap 1: Game Design

We beginnen waar elke game begint: een idee. Het is belangrijk dat je een idee hebt hoe je game eruit komt te zien. Wat kan je allemaal doen in je game? Dit noemen we ook wel Game Design.

Boter kaas en eieren als spel is erg simpel. In deze les ga je de simpelste versie van het spel opzetten. Als je extra elementen toe wilt voegen kan je dat natuurlijk altijd doen. Hier zou je verder over na kunnen denken als je wilt.

Tip: Liever goed gejat dan slecht bedacht. Kijk ook eens naar andere spellen (boter kaas en eieren). Hoe zien die eruit? Wat vind je er goed aan, en wat niet?

Stap 2: HTML

Nu je weet hoe je game eruit komt te zien en wat je erin wilt hebben, is het tijd om de HTML tags toe te voegen aan je webpagina. Kijk hierbij goed naar wat je nodig hebt.

Voor boter kaas en eieren heb je niet heel veel elementen nodig. Het is handig om een tekstelement te hebben en een startknop. Daarnaast heb je natuurlijk de 9 knoppen nodig voor boter kaas en eieren. Meer niet!

De knoppen verdeel je in 3 rijen van 3 knoppen. Dit doe je door div te gebruiken voor elke rij. Binnen deze div zet je dan 3 knoppen. 


<!DOCTYPE html>
<html lang="en">
    <head>
        <title>Boter, kaas en eieren</title>

        <link rel="stylesheet" href="css/boterkaaseieren.css">

Stap 4: Javascript

Nu je alle elementen op de juiste plaats hebt, is het tijd voor het programmeren van je game met javascript.

Het is belangrijk om stap voor stap op te bouwen. Als je naar je webpagina gaat met je game, zie je alleen maar een startknop die niks doet als je erop drukt. Laten we daar verandering in brengen!

Maak een function in je javascript bestand die ervoor zorgt dat de goede elementen zichtbaar worden. Gebruik hiervoor display. 


function StartBKE()
{
    document.getElementById("BKERij0").style.display = "Block";
    document.getElementById("BKERij1").style.display = "Block";
    document.getElementById("BKERij2").style.display = "Block";
}

Stap 4: Javascript

Nu is het tijd om de eerste speler een beurt te geven. Maar wie? Je kan ervoor kiezen om speler X altijd te laten starten, maar dat is niet heel eerlijk. Daarom gaan we het willekeurig maken!

Als eerste hebben we een variabele nodig. Deze noemen we ronde. Deze houdt bij wie er aan de beurt is, speler 0 of speler 1. Speler 0 is ook wel speler X, en speler 1 is ook wel speler O.

We gebruiken de formule om de computer een willekeurig getal te laten kiezen (Komt in les 10 ook nog extra aan bod). Als de uitkomst 0 is, is speler 0 (X) aan zet. Variabele ronde wordt dus 0, en de tekst passen we aan. Hieronder gebruiken we een else. Dan mag speler 1 (O) dus beginnen. Hiervoor doen we hetzelfde. 


let ronde = 0;

function StartBKE()
{
    document.getElementById("BKERij0").style.display = "Block";
    document.getElementById("BKERij1").style.display = "Block";
    document.getElementById("BKERij2").style.display = "Block";
    if (Math.floor(Math.random() * 2) == 0)
    {
        ronde = 0;
        document.getElementById("BKETekst").innerHTML = "Speler X is aan zet";
    }
    else
    {
        ronde = 1;
        document.getElementById("BKETekst").innerHTML = "Speler O is aan zet";
    }
}

Stap 4: Javascript

Nu zegt de tekst wie er begint, maar er gebeurt nog niks als we op de knoppen drukken. Laten we daar verschil in maken! Maak de function BKEKnop(knopNummer). Deze function wordt afgespeeld als een van de knoppen wordt ingedrukt.

Het is handig om een nieuwe array aan te maken voor beide spelers. Dit kan door een array in een array te maken. Deze noemen we zetten. Hierin kunnen we zien wie welke vakken heeft veroverd. zetten[0] geeft de vakken van speler 0 (X) en zetten[1] geeft de vakken van speler 1 (O).

Terug naar de nieuwe function. Als ronde=0 (X is aan de beurt), dan zetten we een X neer in het goede vakje. Ook wordt de nummer van dat vakje toegevoegd aan zetten[0]. Als ronde=1 doen we natuurlijk precies hetzelfde voor speler 1 (O). 


let ronde = 0;
let zetten = [[], []];

function StartBKE()
{
    document.getElementById("BKERij0").style.display = "Block";
    document.getElementById("BKERij1").style.display = "Block";
    document.getElementById("BKERij2").style.display = "Block";
    if (Math.floor(Math.random() * 2) == 0)
    {
        ronde = 0;
        document.getElementById("BKETekst").innerHTML = "Speler X is aan zet";
    }
    else
    {
        ronde = 1;
        document.getElementById("BKETekst").innerHTML = "Speler O is aan zet";
    }
}

function BKEKnop(knopNummer)
{
    if (ronde == 0)
    {
        document.getElementById("BKEKnop" + knopNummer.toString()).innerHTML = "X";
        zetten[0].push(knopNummer);
    }
    else
    {
        document.getElementById("BKEKnop" + knopNummer.toString()).innerHTML = "O";
        zetten[1].push(knopNummer);
    }
}

Stap 4: Javascript

Hij werkt! Het enige probleem is dat hij niet van beurt verandert. Dan wordt het wel heel makkelijk... Tijd om dat onderaan de function in te voegen!

We checken hier opnieuw in welke ronde we zitten. Hier is een goede reden voor, maar die komt iets later aan bod.

Als het nu ronde=0 (X) is, wordt het ronde=1. Ook veranderen we de tekst erboven zodat het klopt.

Nu werkt hij helemaal goed... totdat je op een verkeerde knop drukt. Als de knop al is ingedrukt door, dan gaat jouw beurt verloren. Tijd om daar verandering in te brengen. Bovenaan de function checken we of de knop al is ingedrukt door speler 0 (X) of speler 1 (O). Als dit zo is, gebruiken we return. Dat betekent dat de rest van de function niet meer wordt uitgevoerd, en dat er dus niks gebeurt! 


let ronde = 0;
let zetten = [[], []];

function StartBKE()
{
    document.getElementById("BKERij0").style.display = "Block";
    document.getElementById("BKERij1").style.display = "Block";
    document.getElementById("BKERij2").style.display = "Block";
    if (Math.floor(Math.random() * 2) == 0)
    {
        ronde = 0;
        document.getElementById("BKETekst").innerHTML = "Speler X is aan zet";
    }
    else
    {
        ronde = 1;
        document.getElementById("BKETekst").innerHTML = "Speler O is aan zet";
    }
}

function BKEKnop(knopNummer)
{
    if (zetten[0].includes(knopNummer) || zetten[1].includes(knopNummer))
        return;
    if (ronde == 0)
    {
        document.getElementById("BKEKnop" + knopNummer.toString()).innerHTML = "X";
        zetten[0].push(knopNummer);
    }
    else
    {
        document.getElementById("BKEKnop" + knopNummer.toString()).innerHTML = "O";
        zetten[1].push(knopNummer);
    }
    if (ronde == 0)
    {
        ronde = 1;
        document.getElementById("BKETekst").innerHTML = "Speler O is aan zet";
    }
    else if (ronde == 1)
    {
        ronde = 0;
        document.getElementById("BKETekst").innerHTML = "Speler X is aan zet";
    }
}

Stap 4: Javascript

Hij is bijna klaar, maar hij herkent nog niet wie er gewonnen heeft. Daarvoor gebruiken we een nieuwe function, CheckWin(). Deze roepen we op middenin de BKEKnop() function.

In de CheckWin() function controleren we alle mogelijke combinaties. Vandaar dat de if erg lang lijkt. Deze zou je gewoon over kunnen nemen.

Als zetten[0], dus speler 0 (X) een goede combinatie heeft, dan heeft hij gewonnen! Dat betekent ook dat het spel voorbij is. Voor nu zetten we de variabele ronde dan even op 9. Hier kunnen we later nog meer mee doen. Vervolgens veranderen we de tekst bovenin en de tekst op de knop ook. Dit doen we precies hetzelfde als speler 1 (O) wint.

Maar wat nou als het gelijkspel is? Daarvoor checken we of er in totaal 9 vakjes zijn gevuld. Dit doen we door zetten[0].length en zetten[1].length bij elkaar op te tellen.

Omdat we de variabele ronde 9 hebben gemaakt, wordt er niet een andere speler gekozen om verder te gaan. Het spel is nu dus klaar.

let zetten = [[], []];

function StartBKE()
{
    document.getElementById("BKERij0").style.display = "Block";
    document.getElementById("BKERij1").style.display = "Block";
    document.getElementById("BKERij2").style.display = "Block";
    if (Math.floor(Math.random() * 2) == 0)
    {
        ronde = 0;
        document.getElementById("BKETekst").innerHTML = "Speler X is aan zet";
    }
    else
    {
        ronde = 1;
        document.getElementById("BKETekst").innerHTML = "Speler O is aan zet";
    }
}

function BKEKnop(knopNummer)
{
    if (zetten[0].includes(knopNummer) || zetten[1].includes(knopNummer))
        return;
    if (ronde == 0)
    {
        document.getElementById("BKEKnop" + knopNummer.toString()).innerHTML = "X";
        zetten[0].push(knopNummer);
    }
    else
    {
        document.getElementById("BKEKnop" + knopNummer.toString()).innerHTML = "O";
        zetten[1].push(knopNummer);
    }
    CheckWin();
    if (ronde == 0)
    {
        ronde = 1;
        document.getElementById("BKETekst").innerHTML = "Speler O is aan zet";
    }
    else if (ronde == 1)
    {
        ronde = 0;
        document.getElementById("BKETekst").innerHTML = "Speler X is aan zet";

function CheckWin()
{
    if ((zetten[0].includes(0) && zetten[0].includes(1) && zetten[0].includes(2)) ||
        (zetten[0].includes(3) && zetten[0].includes(4) && zetten[0].includes(5)) ||
        (zetten[0].includes(6) && zetten[0].includes(7) && zetten[0].includes(8)) ||
        (zetten[0].includes(0) && zetten[0].includes(3) && zetten[0].includes(6)) ||
        (zetten[0].includes(1) && zetten[0].includes(4) && zetten[0].includes(7)) ||
        (zetten[0].includes(2) && zetten[0].includes(5) && zetten[0].includes(8)) ||
        (zetten[0].includes(0) && zetten[0].includes(4) && zetten[0].includes(8)) ||
        (zetten[0].includes(2) && zetten[0].includes(4) && zetten[0].includes(6)))
    {
        ronde = 9;
        document.getElementById("BKETekst").innerHTML = "Speler X heeft gewonnen!";
        document.getElementById("startKnopBKE").innerHTML = "Speel opnieuw";
    }
    if ((zetten[1].includes(0) && zetten[1].includes(1) && zetten[1].includes(2)) ||
        (zetten[1].includes(3) && zetten[1].includes(4) && zetten[1].includes(5)) ||
        (zetten[1].includes(6) && zetten[1].includes(7) && zetten[1].includes(8)) ||
        (zetten[1].includes(0) && zetten[1].includes(3) && zetten[1].includes(6)) ||
        (zetten[1].includes(1) && zetten[1].includes(4) && zetten[1].includes(7)) ||
        (zetten[1].includes(2) && zetten[1].includes(5) && zetten[1].includes(8)) ||
        (zetten[1].includes(0) && zetten[1].includes(4) && zetten[1].includes(8)) ||
        (zetten[1].includes(2) && zetten[1].includes(4) && zetten[1].includes(6)))
    {
        ronde = 9;
        document.getElementById("BKETekst").innerHTML = "Speler O heeft gewonnen!";
        document.getElementById("startKnopBKE").innerHTML = "Speel opnieuw";
    }
    if (zetten[0].length + zetten[1].length == 9)
    {
        ronde = 9;
        document.getElementById("BKETekst").innerHTML = "Gelijkspel!";
        document.getElementById("startKnopBKE").innerHTML = "Speel opnieuw";
    }
}

        (zetten[1].includes(1) && zetten[1].includes(4) && zetten[1].includes(7)) ||
        (zetten[1].includes(2) && zetten[1].includes(5) && zetten[1].includes(8)) ||
        (zetten[1].includes(0) && zetten[1].includes(4) && zetten[1].includes(8)) ||
        (zetten[1].includes(2) && zetten[1].includes(4) && zetten[1].includes(6)))
    {
        ronde = 9;
        document.getElementById("BKETekst").innerHTML = "Speler O heeft gewonnen!";
        document.getElementById("startKnopBKE").innerHTML = "Speel opnieuw";
    }
    if (zetten[0].length + zetten[1].length == 9)
    {
        ronde = 9;
        document.getElementById("BKETekst").innerHTML = "Gelijkspel!";
        document.getElementById("startKnopBKE").innerHTML = "Speel opnieuw";
    }
