Les 4: CSS en opmaak


Opmaak met CSS

In deze les ga je leren hoe je je website kan opmaken met de opmaaktaal CSS. CSS werkt anders dan HTML. Volg dus goed de stappen om te kijken hoe het werkt.

Als eerste ga je een CSS bestand aanmaken. Dit werkt hetzelfde als het aanmaken van een HTML bestand, alleen eindigt je bestand met het stukje ".css". Het is handig om een apart mapje aan te maken voor je CSS bestanden. Vaak wordt een CSS bestand styles.css genoemd.

Vervolgens ga je je CSS bestand koppelen aan je HTML bestand (je webpagina dus). Dit doe je door een regel toe te voegen in de "head" van je HTML bestand. Hiernaast zie je een stukje wat je als het goed is in je HTML bestand hebt staan. Voeg de regel toe die er nieuw bijstaat. Het stukje "css/styles.css" kan je vervangen door de juiste map en bestandsnaam. 


<head>
    <title>Titel van je website</title>
    <meta charset="utf-8">
    <link rel="stylesheet" href="css/styles.css"> <!--Deze regel voeg je toe-->
</head>

Tags opmaken

In CSS kan je tags makkelijk opmaken. Dit doe je door properties, oftewel eigenschappen, toe te kennen aan tags, classes of id's. Als eerste gaan we de achtergrond kleur van de tekst veranderen.

Ga naar je CSS bestand. Als het goed is is deze nog leeg. Vervolgens zet je op de eerste regel een tag neer. Dit is de tag die je op gaat maken. In het voorbeeld hiernaast is gekozen voor de h1 tag. Vervolgens druk je enter, en gebruik een accolade (deze -> { }). Druk daarna weer enter.

Nu zit je als het goed is in het stuk tussen de accolades. Hier ga je properties toekennen. Om de achtergrond kleur te veranderen, gebruiken we de property "background-color:". Kies een kleur (engelstalig) en eindig met een puntkomma (deze -> ;). Zie je op je pagina dat de kleur verandert? 


h1
{
    background-color: blue;
}

Classes

Je hebt nu geleerd hoe je een tag op kan maken, maar soms wil je dat de tag h1 de ene keer blauw is, en de andere keer groen. Of je wilt dat de h1 tags en de p tags allemaal een roze achtergrond krijgen. Daarvoor gebruiken we classes.

Als eerste moet je een HTML tag een class toekennen. Om dit te doen ga je naar je HTML bestand. Tussen de hoekhaken (deze -> < >) heb je de naam van de tag staan, bijvoorbeeld h1. Nog tussen de hoekhaken maar na de tag, kan je class="" schrijven. Tussen de aanhalingstekens kan je vervolgens de naam van de class schrijven. Deze naam mag je zelf kiezen, maar kies iets duidelijks. Je kan dezelfde class aan meerdere HTML tags toekennen, ook als de tag niet hetzelfde is!

Vervolgens kunnen we de class in het CSS bestand opmaken. Dit doe je op dezelfde manier als het opmaken van een tag, maar je gebruikt een punt voor de naam van de class. Kan jij de classes toekennen aan je HTML tags en de achtergrond kleur veranderen? 


<h1 class="standaardtekst">Hallo!</h1>
<p class="standaardtekst">daar...</p>

-------------------------------

.standaardTekst
{
    background-color: blue;
}

id's

Je kan ook id's toekennen aan HTML tags op dezelfde manier als classes. Dit kan handig zijn als je binnen een class onderscheid wilt maken tussen tags.

Id's moet je toekennen aan HTML tags. Dit doe je op bijna dezelfde manier als classes. Je gebruikt niet class="", maar je gebruikt id="".

In CSS werkt het ook weer bijna hetzelfde. Het enige verschil is dat je bij een id begint met een hashtag (deze -> #) in plaats van een punt. 


<h1 class="standaardTekst" id="titel">Hallo!</h1>
<p class="standaardTekst">daar...</p>

-------------------------------

.standaardTekst
{
    background-color: blue;
}
#titel
{
    font-size: 40px;
}

Extra: Individuele tags opmaken

Soms is het handig om in 1 geval iets te veranderen aan de opmaak. Als je dit zou willen doen, heb je het CSS bestand niet nodig.

Je kan net na de tag het volgende toevoegen: style="". Vervolgens kan je tussen de aanhalingstekens de opmaak zetten die je nodig hebt. Met een puntkomma (;) onderscheid je verschillende CSS properties.

Let op! Er is een CSS bestand voor een reden. Het kan snel rommelig worden als je je CSS in je HTML gaat vermelden. Probeer dit daarom zo min mogelijk te doen! 


<h1 style="background-color: blue;">Hallo!</h1>

Opdracht: Tags, classes en id's: welke gaat nou voor?

Je hebt nu 4 verschillende manieren geleerd om CSS properties toe te voegen aan tekst, maar wat nou als je ze allemaal gebruikt? Welke van de 4 methoden heeft voorrang, en welke daarna? Probeer hier voor jezelf eens achter te komen!

CSS properties

Nu weet je wat classes en id's zijn, en hoe je CSS properties kan toevoegen. Nu is het tijd om eens te kijken naar een aantal CSS properties die je toe kan voegen aan jouw tekst!

CSS property: background-color

Je hebt al kennis gemaakt met de property background-color, maar wat kan je er verder mee? Je kan een (engelstalige) kleur toevoegen, maar je kan ook een rgb code toevoegen. Een rgb code staat ook voor een kleur. Als je in Visual Studio Code op de kleur klikt, kan je de kleur handmatig aanpassen! 


.cssVoorbeeld
{
    background-color: rgb(11, 11, 177);
}

CSS property: color

Naast background-color, is er ook nog color. Deze property geeft de kleur van de tekst aan. Je kan de kleur op dezelfde manier aangeven als bij background-color. 


.cssVoorbeeld
{
    color: rgb(11, 11, 177);
}

Extra: Geheimschrift

Als je de color en background-color properties van een tag dezelfde kleur maakt, dan maak je geheimschrift. Je kan de tekst namelijk niet lezen! De enige manier waarop je de tekst wel kan lezen, is door de tekst te selecteren.

Goed gedaan! Je hebt geheimschrift gelezen!


.cssVoorbeeld
{
    background-color: rgb(11, 11, 177);
    color: rgb(11, 11, 177);
}

CSS property: font-size

Met de property font-size kan je de lettergrootte aanpassen. Je moet hier wel de eenheid bijzetten. Vaak doe je dit in px (pixels). 


.cssVoorbeeld
{
    font-size: 20px;
}

CSS property: font-family

Je kan ook het lettertype aanpassen. Dit doe je met font-family. Als je dit intoetst in Visual Studio Code, krijg je suggesties van lettertypes. Het lettertype hiernaast wordt vaak gebruikt. 


.cssVoorbeeld
{
    font-family: system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', 
        Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;
}

Opdracht: Maak je tekst mooi!

Gebruik de CSS properties die hierboven beschreven staan om je tekst op te maken!
