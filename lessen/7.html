Les 7: Nieuwe webpagina's


Nieuwe webpagina's aanmaken

In deze les ga je alles leren over nieuwe webpagina's en hoe je van de ene naar de andere webpagina gaat. Daarvoor is het wel belangrijk om te weten hoe je een nieuwe webpagina aanmaakt.

Het aanmaken van een nieuwe webpagina is eigenlijk heel simpel, je hebt het namelijk al eens gedaan. Je gaat naar Visual Studio Code, en je klikt op het icoontje met het bestandteken en een plusje. Let erop dat je nieuwe bestand eindigt met ".html".

Terugblik: Bestandsnamen

Voor het maken van een website zal je veel mapjes en bestanden moeten aanmaken. De namen van deze bestanden zijn bijna nooit van belang, maar het is handig om ze een duidelijke naam te geven. Als je dit niet doet, wordt alles namelijk erg onoverzichtelijk, en weet je niet welk bestand voor wat is.

HTML tag: <a>

Je hebt nu meerdere webpagina's gemaakt, maar hoe kom je nou van de ene naar de andere webpagina? Dat doe je met een hyperlink. Deze kan je aanmaken met de <a> tag.

Zie het voorbeeld hiernaast hoe je de <a> tag gebruikt. Tussen <a> en </a> zet je het element waar je op moet drukken. Dit kan tekst zijn, maar je kan hier ook een afbeelding of een div plaatsen.

Nu heb je misschien een hyperlink, maar hij verwijst nog nergens naartoe. Om dat te doen moet je href="" tussen de haken zetten (zie voorbeeld hiernaast). Tussen de aanhalingstekens kan je de link van een andere website zetten, of je kan het bestandspad van je huidige HTML bestand naar het nieuwe HTML bestand neerzetten.

Tip: Weet je niet goed hoe je een bestandspad moet vermelden of vind je dit nog lastig? Check extra les 1 over bestandspaden! 


<a href="index.html"><img src="img/afbeelding.png"></a>
<a href="meesterthomas.github.io">Hallo, deze tekst is een hyperlink!</a>

Verwijzen op dezelfde pagina

Je kan ook verwijzen naar een element op dezelfde webpagina. Om dit te doen moet je het element waar je naar wilt verwijzen een id geven. Vervolgens zet je bij href de id van het element en daarvoor een hashtag. Zie het voorbeeld hiernaast.

Openen in nieuw tabblad

Als je wilt dat een link wordt geopend in een nieuw tabblad, zodat het huidige tabblad niet verloren gaat, kan dat ook! Daarvoor zet je target="_blank" tussen de haken. Zo simpel is het!

Klik hier om te openen in nieuw tabblad! 


<a href="../index.html" target="_blank">Klik hier om te openen in nieuw tabblad!</a>

Linkjes opmaken

Linkjes kan je niet zo opmaken als andere tekst. Om alle linkjes aan te passen, kan je in CSS de tag a gebruiken om aan te passen hoe een link wordt vormgegeven.

Je kan de a tag op meerdere momenten aanpassen. Zo kan je de normale kleur van de link aanpassen door :link erachter te zetten. Door :visited achter a te zetten, kan je de opmaak van de link aanpassen als de gebruiker de link al een keer heeft bezocht. Je kan zelfs de opmaak veranderen als de gebruiker met de muis over de link zweeft! Dit doe je door :hover erachter te zetten.

Deze verschillende fasen van a kan je ook los van elkaar opmaken. Zie daarvoor het voorbeeld hiernaast.

Tip: Je kan elk element een andere opmaak geven als je eroverheen zweeft met je muis. Hiervoor moet je in CSS :hover toevoegen, net zoals hoe je dat bij het a element doet! 


a
{
    text-decoration: none; <!-- Hiermee haal ik de streep onder een linkje weg-->
}
a:link
{
    color: green;
}
a:visited
{
    color: red;
}
a:hover
{
    color: yellow;
    background-color: blue;
}

Opdracht: Maak je nieuwe webpagina's aan en verwijs ernaar!

Nu weet je hoe je een nieuwe webpagina aan moet maken, en op welke manieren je naar andere webpagina's gaat verwijzen. Bedenk voor jezelf het volgende:
- Hoeveel webpagina's wil je hebben en wat komt er op elke pagina te staan?
- Hoe ga je naar de verschillende webpagina's verwijzen?

Denk over deze punten na, en verwerk ze alvast in je website. Maak de goede webpagina's aan en kijk eens of je ze al mooi kan maken.

Tip: Sommige websites gebruiken een navigatiebalk boven of onder in het scherm of op de webpagina. Deze kan jij ook maken! Bekijk daarvoor extra les 3.
