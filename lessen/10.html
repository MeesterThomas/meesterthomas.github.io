Nieuwe javascript code

In deze les krijg je meer te weten over hoe je javascript in kan zetten. De voorbeelden in deze les zijn soms wat specifieker of ingewikkelder dan die in les 9. Wel kunnen ze heel goed helpen om voor elkaar te krijgen wat je op je website wilt.

Javascript code: return

Als je wilt dat de rest van de code in je function niet wordt uitgevoerd (bijvoorbeeld bij een if statement), dan kan je return gebruiken. Met return stop je de function.

Je kan ook een function aanmaken om informatie te krijgen. Een function die een stuk tekst terug geeft bijvoorbeeld. Als je zo'n function wilt gebruiken, moet je de function eindigen met een return, gevolgd door de informatie die je terug wilt krijgen. Een voorbeeld hiervan is in het codeboek te zien. 


let teller = 0;

function ZingKnop()
{
    teller += 1;
    if (teller >= 5)
        return; //De function stopt hier als het lied al voorbij is.
    document.getElementById("scoreTekst").innerHTML = KrijgLiedTekst();
}

function KrijgLiedTekst() //Deze function geeft tekst terug bij return.
{
    if (teller == 1 || teller == 2 || teller = 4)
        return "Hoofd, schouders, knie en teen, knie en teen!";
    else if (teller == 3)
        return "Oren, ogen, puntje van je neus!";
    else
        return "";
}

Javascript code: array

Je weet inmiddels wat variabelen zijn. Soms wil je misschien meerdere variabelen in een lijst hebben staan, bijvoorbeeld een lijst met quizvragen. Hiervoor kan je een array gebruiken.

Een array is eigenlijk gewoon een lijst van variabelen. Zie het voorbeeld hiernaast hoe je de variabele uit een array moet veranderen.

- Wil je weten hoe lang een array is? Gebruik dan arrayNaam.length.
- Wil je een nieuwe variabele toevoegen aan het einde van de array? Gebruik dan arrayNaam.push(waardeVanNieuweVariabele).
- Wil je een variabele uit een array verwijderen? Gebruik arrayNaam.pop() als je de laatste variabele wilt verwijderen, en arrayNaam.shift() als je de eerste variabele wilt verwijderen. 


let score = [0, 2];

function Doelpunt(team)
{
    document.getElementById("team0Score").innerHTML = score[0]; //"0"
    document.getElementById("team1Score").innerHTML = score[1]; //"2"
    if (team == 0)
        score[0] += 1;
    if (team == 1)
        score[1] += 1;
}

Loops

Loops kunnen handig zijn om bepaalde code meerdere keren te herhalen. Hiervoor kan je ook een bepaalde variabele elke keer veranderen. Loops kunnen lastig zijn om te begrijpen, dus lees goed waar je elke loop voor kunt gebruiken.

Als loops niet goed worden toegepast, kan een bepaald stukje code oneindig vaak worden herhaald, wat ertoe leidt dat je Chromebook vastloopt. Check je code bij een loop dus altijd extra goed! Mocht je Chromebook vastlopen, sluit dan het tabblad van je webpagina. Als dat niet gaat, moet je je Chromebook opnieuw opstarten.

Javascript code: loops met while

Een while loop loopt door zijn code heen zolang een bepaalde voorwaarde waar is. Eigenlijk werkt het hetzelfde als een if statement, maar dan loopt hij door zijn code heen.

Een while loop eindigt niet uit zichzelf. Het is belangrijk dat je dit zelf in de loop verwerkt. Misschien door een teller omhoog te laten gaan als de code wordt doorlopen. 


while (nummer <= 10)
{
    CodeOmUitTeVoeren();
    nummer += 1;
}

Javascript code: loops met for

Een for loop is al iets veiliger dan een while loop, maar heeft meer regels om rekening mee te houden.

Een for loop heeft zelf een ingebouwde teller die je benoemt aan het begin van de loop. Tussen de haakjes komen 3 onderdelen te staan. De onderdelen worden gescheiden door een ;.
- Bij het eerste onderdeel wordt een nieuwe variabele als teller benoemd. In het voorbeeld is dat i.
- Bij het tweede onderdeel wordt aangegeven hoelang de loop door blijft gaan. In het voorbeeld is dat zolang de teller i onder de lengte van de array blijft.
- Bij het derde onderdeel wordt aangegeven wat er gebeurt na 1 loop. In het voorbeeld komt er 1 bij de teller i erbij. Op die manier gaat de loop dus niet oneindig door.

Vaak wordt een for loop gebruikt op de manier van het voorbeeld hiernaast. 


for (let i = 0; i < array.length; i++) 
{
    CodeOmUitTeVoeren();
}

Javascript code: loops met forEach

Een forEach loop kan alleen gebruikt worden bij een array. Deze loop loopt namelijk door alle elementen van een array. Je zal dus ook niet zo snel problemen krijgen omdat de computer oneindig vaak loopt.

Je ziet in het voorbeeld hiernaast hoe je een forEach loop aangeeft. Het stukje array moet je vervangen met de naam van de array. Het stukje element kan je zelf een naam geven. Dit wordt de variabele van het huidige element van de array. 


array.forEach(element => 
{
    CodeOmUitTeVoeren();
});

Terugblik: Soorten variabelen

Variabelen kunnen ook meerdere soorten waardes hebben. Denk hierbij aan wat er in het doosje zit. Dit zijn de 3 belangrijkste:
- Een "string" oftewel een stuk tekst. Dit kan zo lang of kort zijn als je maar wilt. Let op: een string zet je altijd tussen ""!
- Een nummer. Deze spreekt voor zich. Met nummers kan je de computer laten rekenen. Let op: als je nummers met decimale getallen gebruikt, moet je een . gebruiken in plaats van een ,!
- Een bool (of boolean). Dit is een variabele die true (goed) is, of false (fout).

Javascript code: toString()

Als een variabele een nummer of een bool is, kan je hem soms niet gelijk als tekst op je website zetten. Als dit niet lukt, kan je .toString() erachter zetten. Hiermee wordt de waarde een stukje tekst!. 


score.toString();

Javascript code: parseInt()

Als je de waarde van een string (een stuk tekst) om wilt zetten naar een nummer, bijvoorbeeld om ermee te rekenen, kan je parseInt(string) gebruiken. Het stukje string moet je dan vervangen met het stukje tekst. 


parseInt(scoreTekst);

Javascript code: Math.

Met Math. kan je verschillende dingen doen die te maken hebben met rekenen en wiskunde. Er zijn een paar voorbeelden die we hier behandelen.

Als je een kommagetal wilt afronden, kan je de volgende gebruiken:
- Math.round(getal) rond het getal tussen de haakjes af volgens de normale afrondingsregels.
- Math.ceil(getal) rond het getal tussen de haakjes altijd naar boven af (1.0001 wordt 2).
- Math.floor(getal) rond het getal tussen de haakjes altijd naar beneden af (1.9999 wordt 1).

Ook kan je een willekeurig getal gebruiken met Math.random(). De computer genereert een getal van 0 tot 1. 0 is wel mogelijk, maar 1 niet.

Let op: In code gebruik je een punt om een decimaal getal (kommagetal) aan te wijzen. Je zegt dus 1.5 en niet 1,5. 


let getal = 1.5;
Math.round(getal); //Dit wordt 2
Math.ceil(getal); //Dit wordt 2
Math.floor(getal); //Dit wordt 1
getal = Math.random; //Nu is getal een willekeurig getal tussen de 0 en 1

Extra: Willekeurige optie kiezen

Nu je weet hoe je een willekeurig getal kan aanmaken, kan je ook de computer een willekeurige keuze laten maken. Hiervoor kan je de code gebruiken die hiernaast staat.

We hebben hiermee een variabele genaamd "willekeurig" aangemaakt. Deze variabele gaat een getal krijgen. Je ziet dat hij een willekeurig getal krijgt, en dat deze vermenigvuldigd wordt met het aantal opties. Door dit vervolgens af te ronden naar beneden, wordt "willekeurig" een willekeurig getal tussen de 0 en de waarde van aantalOpties.

Vervolgens kan je met if statements voor elke mogelijke (hele) waarde van willekeurig een uitkomst beschrijven in code. 


let willekeurig = Math.floor(Math.random() * aantalOpties);

Javascript code: .split

Met .split() kan je van een variabele een array maken door hem in stukjes te hakken. Tussen de haakjes geef je aan bij welk teken de tekst in stukken wordt gehakt. Let op dat je dit gebruikt om een array variabele te vullen. 


arrayNaam = vragenLijst.split('?');

Javascript code: .includes

Als je wilt weten of een array een bepaalde waarde bevat, of als je wilt weten of er bepaalde tekst binnen een tekst variabele staat, kan je string.includes("tekst") gebruiken. Het stukje string moet je vervangen door de naam van de array, of de naam van de string die je wilt onderzoeken. Het stukje "tekst" tussen de haakjes moet je vervangen door de tekst die je zoekt. Dit geeft true of false aan, dus dit kan je gebruiken in een if statement. 


if (teamNamen.includes("Blauw-Wit"))
{
    codeOmUitTeVoeren();
}
else if (teamNamen[0].includes(":"))
{
    codeOmUitTeVoeren();
}

Extra: console.log

Let op: Dit kan je niet doen op de Chromebook van school, aangezien deze beveiligd is voor deze functie. Dit moet je dus thuis proberen op een andere laptop of computer.

Soms wil je eens zien welke waarde een variabele heeft binnen je code, bijvoorbeeld als er iets niet werkt hoe het hoort. Daarvoor kan je console.log() gebruiken. Tussen de haakjes kan je de variabele zetten die je wilt weten. Als je de website vervolgens de code laat uitvoeren, kan je via inspecteren de waarde van de variabele zien in de console (onderaan). 


console.log(score);
