Les 6: Extra CSS


CSS property: display

Met de display property kan je de opmaak op de pagina van een HTML element regelen. Er zijn meerdere opties voor de display van een element.

"display: block;" is voor de meeste tags de standaard. Dit betekent dat een HTML tag (als dat mogelijk is) een hele regel in beslag neemt, en dat de tag dus altijd begint op een nieuwe regel.

"display: inline;" is het tegenovergestelde. Dit betekent dat een HTML tag midden in een zin wordt toegevoegd, en dus niet een nieuwe regel in beslag neemt.

"display: flex;" zit er een beetje tussenin. Zoals de naam al zegt, wordt het element flexibel met deze property. Alle elementen binnen dit element worden op 1 regel gepropt, maakt niet uit hoeveel het er zijn!

Deze property kan je dus aan een element toevoegen. Een <p> tag bijvoorbeeld, is van zichzelf een block-element. Maar, als je de property "display: inline;" toevoegt, dan wordt het een "inline" element. 


.cssVoorbeeld
{
    background-color: blue;
    display: block;
}
.cssVoorbeeld
{
    background-color: blue;
    display: inline;
}
.cssVoorbeeld
{
    background-color: blue;
    display: flex;
}

Extra: Ouders en kinderen

In een HTML bestand heb je te maken met tags binnen een andere tag. Om deze te omschrijven, wordt erg verwarrend. Vandaar dat we deze tags met een relatie tot elkaar "parents" (ouders) en "children" (kinderen) noemen. Een tag binnen een andere tag, is een child, en het element waar de child in zit, is de parent. 


<div id="parent"> <!--Dit is de parent van "child1" en "child2"-->
    <div id="child1"> <!--Dit is de child van "parent" en de parent van de 2 p tags-->
        <p>Hier staat tekst</p>
        <p>Hier staat nog meer tekst</p>
    </div>
    <div id="child2"> <!--Dit is de child van "parent" en de parent van de 2 p tags-->
        <p>Hier staat tekst</p>
        <p>Hier staat nog meer tekst</p>
    </div>
</div>

Opdracht: Laat display zien!

Je hebt gelezen over de 3 belangrijkste vormen van de display property. Probeer ze eens uit! Kijk wat je kan met alle 3 de vormen.
Tip: Geef de verschillende elementen met de display property een andere achtergrondkleur om makkelijker het verschil te zien.

CSS properties: justify-content en align-items

Als je "display: flex;" hebt gebruikt voor een element, kan het zijn dat je child elementen niet op de goede plaats terechtkomen. Dit gebeurt vooral als je child element een vaste width of height hebben.

Met "justify-content" kan je de horizontale positie bepalen. Dit kan zijn "center", "left" of "right".

Met "align-items" kan je de verticale positie bepalen. Dit kan zijn "center", "start" en "end".

Als je deze properties nog niet helemaal snapt, probeer ze eens uit. Let op: de parent moet de "display: flex;" property en de bovenstaande properties hebben. Het gaat dan om de positie van het child element. 


.cssVoorbeeld
{
    background-color: blue;
    display: flex;
    justify-content: center;
}
.cssVoorbeeld
{
    background-color: blue;
    display: flex;
    align-items: center;
}

CSS property: position

Met de position property kan je de positie van een element bepalen. Er zijn verschillende mogelijkheden:

Met "position: static;" wordt een element op de normale manier op de webpagina geplaatst. Dit is de standaard voor alle HTML tags.

Met "position: relative;" wordt een element ook op de normale manier op de webpagina geplaatst, maar deze kan je vervolgens aanpassen met de CSS properties left, right, top en bottom. Daarmee kan je dus de positie een klein beetje aanpassen.

Met "position: fixed;" wordt de positie van een element relatief gezien van het scherm. Dit betekent dat het element altijd op het scherm blijft staan, ook als je gaat scrollen op de webpagina. De positie kan je aanpassen met de CSS properties left, right, top en bottom.

Met "position: absolute;" wordt een element op de webpagina geplaatst relatief gezien van de parent van het element. Het element wordt dus in het parent element geplaatst. De positie binnen het parent element kan je aanpassen met de CSS properties left, right, top en bottom. Het parent element kan niet "position: static;" hebben.

Met "position: sticky;" wordt een element op dezelfde manier op de webpagina gezet als bij "position: relative;". Het enige verschil is dat het element zich vervolgens gaat gedragen als "position: fixed" als je gaat scrollen. Hiervoor moet je minimaal een van de volgende CSS properties gebruiken, anders werkt het niet: left, right, top en bottom. 


.cssVoorbeeld
{
    background-color: blue;
    position: relative;
    left: 20%;
}
.cssVoorbeeld
{
    background-color: green;
    position: fixed;
    bottom: 0;
    right: 0;
}
.cssVoorbeeld
{
    background-color: red;
    position: absolute;
    bottom: 0;
    left: 10%;
}
.cssVoorbeeld
{
    background-color: yellow;
    position: sticky;
    top: 0;
}

CSS property: float

Met float kan je een element laten "zweven" aan de linker of rechterkant van het element. Dit kan ervoor zorgen dat tekst of "display: inline;" elementen er rondom uitkomen.

Je hebt "float: left;" en "float: right;" hiervoor die een element aan de linker- of rechterkant laten "zweven". 


.cssVoorbeeld
{
    background-color: blue;
    float: left;
}
.cssVoorbeeld
{
    background-color: blue;
    float: right;
}

Extra CSS property: background-image

Je hebt nu al vaak genoeg geoefend met de property background-color, maar wist je dat je je achtergrond ook een foto kan maken? Dat kan je doen met de property "background-image". Zie het voorbeeld hiernaast hoe het werkt.

Let op: De link naar je bestand kan anders zijn. Let er goed op dat je op de juiste manier naar je bestand verwijst! 


.cssVoorbeeld
{
    background-image: url("../img/backgroundImage.png"); 
}
