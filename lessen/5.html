Les 5: Het box model


Wat is het box model?

Tot nu toe hebben we een aantal CSS properties behandeld om de opmaak van je website vorm te geven. Vandaag ga je leren over het box model. Dit gaat je helpen om je website er echt netjes uit te laten zien!

Het box model bestaat uit een aantal elementen. "Content" is de inhoud van je element HTML tag, dus een afbeelding of een stuk tekst. "Border" is de rand rondom deze tag. "Padding" is de ruimte tussen je content en de border. Tot slot is "Margin" de afstand van de tag tot een andere tag, dus rondom de border.

CSS properties: width en height

De content van je tag kan je dus groter en kleiner maken met de properties width en height. Deze properties zie je het beste terug als je tag ook een achtergrond kleur heeft. Je kan de grootte in meerdere vormen aangeven, waaronder het percentage van het scherm, en de grootte in pixels (px). 


.cssVoorbeeld
{
    background-color: blue;
    width: 100%;
    height: 50px;
}

CSS properties: border-width, border-style, border-color en border-radius

Zonder CSS te gebruiken, heeft een tekst tag geen border (rand). Deze kan je toevoegen met de property border-width. Deze kan je weer aangeven in percentage of in pixels.

Je hebt ook een border-style nodig om aan te geven wat voor patroon de border zal krijgen. Als je een standaard border wilt, gebruik je "solid". Kijk eens welke opties Visual Studio Code heeft en probeer ze eens uit!

Vervolgens kan je je border een kleur geven met border-color. Dit werkt hetzelfde als het kiezen van een achtergrond kleur.

Tot slot kan je de hoeken van je border rond maken. Dit doe je met border-radius. Probeer maar eens te kijken hoe rond je de hoeken kan maken! 


.cssVoorbeeld
{
    background-color: blue;
    border-width: 5px;
    border-style: solid;
    border-color: grey;
    border-radius: 5px;
}

CSS property: padding

Als je de afstand van je tekst tot de border groter wilt maken, kan je de padding aanpassen. Je kan dit op meerdere manieren doen.
- Als je maar 1 maat invult, dan is dit de padding aan alle 4 de kanten.
- Als je 2 maten invult, is de eerste maat voor de padding aan de boven- en onderkant, en de tweede voor de linker- en rechterkant.
- Als je 4 maten invult, is de eerste voor de bovenkant, de tweede voor de rechterkant, de derde voor de onderkant en de vierde voor de linkerkant. De maten gaan dus met de klok mee.

Als je maar 1 deel van de padding wilt aanpassen, kan je ook de properties padding-top, padding-bottom, padding-left en padding-right gebruiken. 


.cssVoorbeeld
{
    background-color: blue;
    border-width: 5px;
    border-style: solid;
    padding: 10px 5px 2px 8px;
}

CSS property: margin

De margin van een tag werkt precies hetzelfde als de padding, maar dan gaat het om de ruimte buiten de border. Hoe je deze benoemt is hetzelfde als padding.
- Als je maar 1 maat invult, dan is dit de margin aan alle 4 de kanten.
- Als je 2 maten invult, is de eerste maat voor de margin aan de boven- en onderkant, en de tweede voor de linker- en rechterkant.
- Als je 4 maten invult, is de eerste voor de bovenkant, de tweede voor de rechterkant, de derde voor de onderkant en de vierde voor de linkerkant. De maten gaan dus met de klok mee.

Als je maar 1 deel van de margin wilt aanpassen, kan je ook de properties margin-top, margin-bottom, margin-left en margin-right gebruiken. 


.cssVoorbeeld
{
    background-color: blue;
    border-width: 5px;
    border-style: solid;
    margin: 10px 5px 2px 8px;
}

CSS property: text-align

Nu je het box model gebruikt, wil je misschien dat je tekst in het midden van de box komt te staan. Dat kan met de CSS property text-align! Je maakt hem vervolgens "left" als je de tekst aan de linkerkant wilt, "right" als je de tekst aan de rechterkant wilt of "center" als je de tekst in het midden wilt hebben. 


.cssVoorbeeld
{
    background-color: blue;
    border-width: 5px;
    border-style: solid;
    text-align: center;
}

Opdracht: Maak een nette website met het box model!
