Les 2: Tekst, titels en kopjes


HTML Tags

Pak je HTML bestand van de vorige les erbij. Een HTML document is de basis voor je website. In dit document komt alles te staan wat je op je website wilt zetten, van tekst tot plaatjes en video's. Deze elementen die je toe kan voegen hebben een bepaalde HTML "tag". Een HTML tag is een bepaalde code die aan je chromebook doorgeeft wat je op de website te zien krijgt. De tags die je gebruikt om je website op te bouwen zet je bijna altijd in de <Body> tag.

Een HTML tag begint altijd met een < en eindigt met een >. In deze les zal je verschillende tags tegenkomen die je kan gebruiken voor je website.

Geordend werken

Nu zal je HTML bestand nog kort zijn, maar als je nieuwe tags toe gaat voegen kan het bestand al snel langer worden. Als er zoveel staat, wordt het lastig om de het bestand te lezen. Daarom zijn er een paar belangrijke regels om het overzicht te houden. Hier komt een zo'n regel.

Veel tags zien er zo uit: <tag>inhoud</tag>. Soms kan de inhoud ertussen op 1 regel blijven staan, zoals bij tekst. Als dit niet gaat, dan is het gebruikelijk om de inhoud op een aparte regel te plaatsen en hem een stukje te laten inspringen met tab, zoals hiernaast:


<tag>
    inhoud
</tag>

HTML Tags: <header>, <div> en <footer>

De tags header, div en footer helpen je om overzicht te houden in je HTML bestand. De header is vaak het bovenste deel van een webpagina, en een footer is het onderste deel. Een div kan je gebruiken om een bepaald stuk van je webpagina aan te geven. Je kan meerdere div tags gebruiken op je webpagina. Je kan zelfs een div gebruiken in een div! Je moet deze tags zien als de inleiding, de alinea's en het slot van een tekst.


<header>
    <p>Dit is de header van de website.</p>
</header>
<div>
    <p>Dit is een div van de website.</p>
</div>
<div>
    <p>Dit is een div van de website.</p>
</div>
<footer>
    <p>Dit is de footer van de website.</p>
</footer>

HTML Tag: <p>

De p tag wordt gebruikt voor simpele tekst op je website. Je kan de tekst zo lang maken als je zelf wilt.


<p>
    Hier komt simpele tekst te staan.
</p>

Extra: kopiëren en plakken

Of ze het nou toe willen geven of niet, iedereen kopieert weleens iets van een ander en daar is niks mis mee! Ook bij het maken van een website kan het enorm waardevol zijn om stukken code te kopiëren van de ene plaats en te plakken op de andere plaats. Soms wil je de volgorde van je webpagina veranderen, ook daarvoor kan het heel nuttig zijn. Kopieer en plak dus vooral als dat nodig is voor jouw website!

Je Chromebook denkt gelukkig met je mee en maakt het makkelijk om dit te doen. Linksonder op je toetsenbord zie je de ctrl (control) toets staan. Deze heb je ervoor nodig. Als je een stuk tekst selecteert, kan je deze kopiëren door tegelijkertijd op "ctrl" en "c" te drukken. Vervolgens kan je het ergens plakken met "ctrl" en "v".

Als je net iets hebt aangepast wat je wilt terug kan draaien, kan dat gelukkig ook! Het terugdraaien van je vorige actie kan met "ctrl" en "z". Als je te ver bent teruggegaan, of je wilt toch weer vooruit, kan je "ctrl" en "y" gebruiken. Let op: als je iets ongedaan hebt gemaakt met ctrl z en vervolgens verander je iets, kan je geen ctrl y meer gebruiken.

HTML Tags: <h1>, <h2>, <h3>, <h4>, <h5> en <h6>

De h tags worden gebruikt als titels en tussenkopjes. h1 is de grootste en h6 is de kleinste.


<h1>
    Deze tekst is groot.
</h1>
<h2>
    Deze tekst is iets kleiner.
</h2>
<h3>
    Deze tekst is weer kleiner.
</h3>
<h4>
    En zo wordt deze tekst weer kleiner...
</h4>
<h5>
    En kleiner...
</h5>
<h6>
    En kleiner...
</h6>

HTML Tag: <br>

De br tag wordt gebruikt als "enter". Na een br tag gaat de tekst verder op de volgende regel. Deze tag wordt niet afgesloten met </br>, alleen <br> is genoeg.


<p>
    Hier komt simpele tekst te staan.<br>En dit komt op de volgende regel.
</p>

Extra: Opmerkingen

Soms is het handig om een opmerking te zetten in je HTML website om overzicht te geven. Dit kan bijvoorbeeld zijn om duidelijk te maken waar een stukje code voor is. Dit kan je doen met opmerkingen. Opmerkingen hebben geen invloed op de webpagina zelf en zie je alleen terug in de code. Opmerkingen begin je met <!-- en eindig je met -->.


<p> <!--Dit is een opmerking. Hier staat simpele tekst.-->
    Hier komt simpele tekst te staan.
</p>

Opdracht: Gebruik de tags

Probeer de tags die hierboven beschreven staan te gebruiken in je website. Probeer alle h tags eens uit en zet ze onder elkaar. Zie je het verschil?

Extra: De betekenis van de tags

Hier komen de tags die voorbij zijn gekomen met hun betekenis:
Header: kop(tekst) in het Nederlands.
Div: division, of in het Nederlands verdeling.
Footer: voet(tekst) in het Nederlands.
P: paragraph, of in het Nederlands paragraaf.
H1 t/m h6: heading, of in het Nederlands (tussen)kopje.
Br: break, of in het Nederlands afbreken (van de zin).
