Les 8: Introductie javascript


Programmeren met javascript

In deze les ga je leren hoe je kan programmeren met javascript. Javascript is een programmeertaal die wij gaan gebruiken bij het maken van een website.

Net zoals bij CSS, gaan we de javascript code in een apart bestand zetten. Dit wordt een javascript bestand. Het is handig om een apart mapje aan te maken voor je javascript bestanden (vaak genaamd "js"). Vaak wordt een javascript bestand script.js genoemd.

Nu je een javascript bestand hebt, moet je deze nog linken aan je HTML bestand(en). Dit doe je met een script tag, net zoals hiernaast is aangegeven. Het stukje "js/script.js" kan je vervangen door het juiste bestandspad.

Tip: Weet je niet goed hoe je een bestandspad moet vermelden of vind je dit nog lastig? Check extra les 1 over bestandspaden! 


<head>
    <title>Titel van je website</title>
    <meta charset="utf-8">
    <link rel="stylesheet" href="css/styles.css"> 
    <script src="js/script.js"></script> <!--Deze regel voeg je toe-->
</head>

Wat is programmeren?

Bij programmeren geef je de computer een stappenplan om uit te voeren. Dit stappenplan schrijf je in Javascript. In deze les ga je vooral leren hoe je de computer bepaalde taken uit kan laten voeren. Dit is de simpelste versie van programmeren. We beginnen met het ervoor zorgen dat de computer een stappenplan gaat volgen als er op een knop wordt gedrukt.

HTML tag: <button>

Deze HTML tag staat voor een knop op je webpagina. Van zichzelf doet de knop nog niks, maar daar kan je met Javascript aanpassen.

De tekst binnen een button tag staat voor de tekst die op de knop komt te staan. Met CSS kan je het uiterlijk van de button verder aanpassen. Dit heb je al eerder geleerd.

Tussen de haakjes kan je ook een "onclick" element toevoegen. Deze hebben we later nodig om de knop iets te laten doen als je erop klikt. 


<button onclick="">Deze tekst komt op de knop te staan!</button>

Javascript code: function

Javascript code is opgebouwd uit een heleboel verschillende functions. Je moet een function zien als een stappenplan wat kan worden uitgevoerd. Je kan meerdere functions of stappenplannen in een javascript bestand.

Om een function te beginnen in javascript, schrijf je function met daarachter de naam van de function. Deze naam mag je zelf verzinnen, maar probeer een duidelijke naam te kiezen die vertelt wat de function doet. De naam van de function sluit je altijd af met (). Later leer je dat je hier wat in kan zetten, maar voor nu is dat niet belangrijk.

Als je die regel hebt, ga je naar de volgende regel met enter, en gebruik je een {. Hiermee open je de function. Je sluit de function weer af met }. Hiertussen komt het stappenplan te staan. 


function DrukKnop()
{
    ;
}

Extra: haakjes

Je zult veel verschillende haakjes moeten gebruiken bij javascript. Allemaal hebben ze een andere functie. Wel is het belangrijk dat je een haakje altijd ook weer afsluit.

(): Deze gebruik je voor de variabele van een function, of bij een if, else if of else statement. Hierover leren jullie meer in les 9.

{}: Deze gebruik de code tussen functions, of de code in een if, else if of else statement. Over het tweede leren jullie meer in les 9.

[]: Deze gebruik je bij arrays. Hierover leren jullie meer in les 10.

Javascript code: document.getElementById()

Met function hebben we een stappenplan aangemaakt, maar welke stappen kunnen we nou eigenlijk meegeven? Er zijn heel veel stappen om mee te geven, maar laten we beginnen met het veranderen van een ander element op de webpagina.

Om een element te veranderen, moet de computer eerst weten welk element dat is. Daarvoor gebruiken we "document.getElementById()". Hiermee kunnen we een element van de webpagina te pakken krijgen met een bepaalde id. Tussen de haakjes zet je tussen "" de naam van het id.

Weet je niet meer hoe je een element een id moet geven? Dit staat uitgelegd in les 4.

We hebben nu het element wat we gaan veranderen, maar de computer moet nog wel weten wat we eraan moeten veranderen. Dat komt nu aan bod! Er zijn een aantal opties.

- Je kan de tekst veranderen. Hiervoor gebruik je ".innerHTML". Vervolgens kan je met = aangeven wat de tekst moet worden (vergeet de tekst niet tussen "" te zetten!). De innerHTML verandert het stuk binnen de begin en eindtags van het element.

- Je kan de CSS van de tekst veranderen. Hiervoor gebruik je ".style." en vervolgens de CSS property die je wilt veranderen. Na de = geef je de nieuwe toestand van de property aan (vergeet de toestand niet tussen "" te zetten!).

Belangrijk: Sluit een opdracht zoals deze in javascript altijd af met een puntkomma (;)! 


function DrukKnop()
{
    document.getElementById("veranderTekst").innerHTML = "De tekst is veranderd!";
    document.getElementById("veranderTekst").style.color = "rgb(11, 11, 177)";
}

Opdracht: Probeer eens een tekst element te veranderen

Je hebt nu geleerd hoe je een element kan veranderen door op een knop te drukken. Probeer het eens uit! Lukt het jou om een p tag van kleur te veranderen of om andere tekst neer te zetten?

Opdracht gemaakt of kom je er niet uit? Bekijk het voorbeeld hieronder!

Voorbeeld van de opdracht

HTML tag: <input>

Misschien wil je dat de gebruiker van je website informatie moet invullen. Dit kan met de input tag. Deze tag heeft geen eindtag. Deze tag kan een heleboel verschillende vormen aannemen. Dit wordt geregeld met type="" tussen de haakjes. Hieronder staan een aantal voorbeelden.

type="text": Hier kan de gebruiker tekst schrijven en invullen. Met minLength="" en maxLength="" kan je het minimaal en maximaal aantal tekens bepalen.

type="checkbox": Dit geeft een checkbox. De gebruiker kan de checkbox waar of niet waar maken door erop te klikken.

type="number": Dit geeft een veld waar de gebruiker een nummer in kan toetsen. Letters en andere symbolen zijn niet toegestaan. met min="" en max="" kan je het minimum en maximum getal kiezen. 


<input type="text" minLength="1" maxLength="10">
<input type="checkbox">
<input type="text" min="0" max="100">

Opdracht: Verander tekst in de ingevoerde tekst

Je weet nu hoe je tekst moet veranderen door op een knop te drukken. Ook weet je hoe je een tekstbox kan maken. Nu gaan we die twee dingen combineren! Hoe kan je ervoor zorgen dat de tekst op de webpagina verandert naar de ingevoerde tekst? Gebruik het voorbeeld hiernaast. Zou jij ook nog de kleur kunnen veranderen? 


<input id="invoerTekst" type="text" minLength="1" maxLength="10">
<button onclick="DrukKnopTekst">Verander tekst!</button>
<p id="veranderTekst">Dit is normale tekst.</p>

-------------------------------

function DrukKnopTekst()
{
    document.getElementById("veranderTekst").innerHTML = document.getElementById("invoerTekst").value;
}
