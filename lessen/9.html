Les 9: Programmeren met javascript


Variabelen

Variabelen zijn ontzettend belangrijk bij programmeren. Variabelen moet je zien als een soort doosje. In het doosje kan van alles zitten en hoeveel ervan als je maar kan bedenken. Deze doosjes bewaart de computer goed, en je kan altijd kijken wat erin zit en soms aanpassen wat er in het doosje zit.

Dit is ontzettend handig in een function, oftewel een stappenplan. Soms wil je alleen een stap laten uitvoeren als iets zo is. Bijvoorbeeld, misschien wil je een quiz maken met punten. Je geeft natuurlijk alleen punten als het antwoord ook goed is. Je kan dus niet standaard de punten uitdelen, maar alleen als het antwoord goed is.

Deze les gaan jullie aan de slag met variabelen en if, else if en else statements. We beginnen met variabelen, de doosjes. Er zijn twee verschillende soorten doosjes, dus laten we eens kijken.

Javascript code: let en const

Variabelen werken op een bepaalde manier binnen een script of javascript bestand. Een variabele bestaat namelijk alleen maar binnen de {} haken waarin hij benoemd is.

Kijk naar het voorbeeld hiernaast. nummer0 wordt benoemd buiten alle {} haken. Dat betekent dat hij bestaat binnen al die {} haken. nummer1 daarentegen, wordt benoemd binnen de {} haken van de function SchrijfNummer(). Buiten deze function bestaat deze variabele dus niet. Daarom geeft voorbeeldTekst3 een error.

Om terug te gaan naar het voorbeeld van de doosjes, laten we zeggen dat een function een kamer is. Het doosje van nummer0 wordt benoemd binnen het bestand, dus binnen het gebouw. Dit doosje staat dus in de berging van het gebouw. Het doosje nummer1 wordt benoemd binnen de kamer SchrijfNummer(). In die kamer staat het doosje dus ook. Dat betekent dat ik niet bij het doosje van nummer1 kan als ik in de kamer van SchrijfNummerOpnieuw() zit.

Variabelen kan je benoemen met let en const.
- Met const kan je een variabele benoemen die constant is, en die dus niet veranderd hoeft te worden. De waarde van een const variabele kan je namelijk ook niet veranderen.
- Met let kan je andere variabelen benoemen. Deze variabelen kan je later nog eens veranderen. Als je dit doet, moet je let er niet voorzetten.

Variabelen kunnen ook meerdere soorten waardes hebben. Denk hierbij aan wat er in het doosje zit. Dit zijn de 3 belangrijkste:
- Een "string" oftewel een stuk tekst. Dit kan zo lang of kort zijn als je maar wilt. Let op: een string zet je altijd tussen ""!
- Een nummer. Deze spreekt voor zich. Met nummers kan je de computer laten rekenen. Let op: als je nummers met decimale getallen gebruikt, moet je een . gebruiken in plaats van een ,!
- Een bool (of boolean). Dit is een variabele die true (goed) is, of false (fout). 


let nummer0 = 0;

function SchrijfNummer()
{
    let nummer1 = 1;
    document.getElementById("voorbeeldTekst0").innerHTML = nummer0;
    document.getElementById("voorbeeldTekst1").innerHTML = nummer1;
    SchrijfNummerOpnieuw();
}

function SchrijfNummerOpnieuw()
{
    document.getElementById("voorbeeldTekst2").innerHTML = nummer0;
    document.getElementById("voorbeeldTekst3").innerHTML = nummer1;
}

//voorbeeldTekst0 = 0
//voorbeeldTekst1 = 1
//voorbeeldTekst2 = 0
//voorbeeldTekst3 = error: bestaat niet!

Variabelen toekennen

Variabelen hebben een bepaalde waarde. Denk aan wat er in die doosjes zit. Deze waarde kan je veranderen. Hiervoor heb je bepaalde manieren.
- =: Met = zeg je de waarde van een variabele. De nieuwe waarde vervangt de oude waarde.
- +=: Hiermee gebruik je de oude waarde, en tel je er een bepaald getal bij op.
- -=: Hiermee gebruik je de oude waarde, en haal je er een bepaald getal vanaf.
- *=: Hiermee gebruik je de oude waarde, en vermenigvuldig je het met een bepaald getal.
- /=: Hiermee gebruik je de oude waarde, en deel je het door een bepaald getal. 


let nummer0 = 0;
nummer0 = 1;
nummer0 += 2;
nummer0 -= 2;
nummer0 *= 1;
nummer0 /= 1;

Javascript code: if, else if en else

Met de if, else if of else statements kan je ervoor zorgen dat maar een deel van een stappenplan wordt gevolgd. Hiernaast zie je een voorbeeld hoe deze statements in je code komen te staan.

Als je een stukje code alleen wilt laten lezen bij een bepaalde voorwaarde, gebruik je if.

Als je if hebt gebruikt, is het mogelijk dat een deel van je code wordt overgeslagen als de voorwaarde niet klopt. Als je een stuk code alleen wilt laten uitvoeren onder een andere voorwaarde, maar alleen als de eerste if wordt overgeslagen, dan gebruik je else if.

Zolang je if of else if statements gebruikt, zal er altijd de mogelijkheid zijn dat geen van de voorwaarde kloppen. Als je dan toch wilt dat er code wordt uitgevoerd, kan je een else statement gebruiken.

Je kan altijd if statements gebruiken wanneer je wilt. Je hebt wel eerst een if statement nodig voordat je een else if statement gebruikt. Als er een if statement is, kan je zoveel else if statements maken als je wil. Voor een else statement is een if statement eerst nodig, maar een else if statement is daar niet voor nodig. 


if (nummer0 == 0)
{
    nummer0 = 1;
}
else if (nummer0 == 1)
{
    nummer0 = 2;
}
else if (nummer0 == 2)
{
    nummer0 = 3;
}
else
{
    nummer0 = 0;
}

Operators

We zijn er bijna doorheen! We hebben geleerd hoe if, else if en else statements werken, maar nu moeten we die statements nog invullen. Dat doen we met operators. Dat klinkt ingewikkeld, maar is eigenlijk heel simpel. Een operator geeft aan wat de relatie moet zijn tussen 2 variabelen.

Javascript code: ==, !=, <, <=, >, >=, &&, ||

==: Hiermee geef je aan dat de waardes aan beide kanten precies hetzelfde moet zijn.
!=: Hiermee geef je aan dat de waardes aan beide kanten juist niet hetzelfde mogen zijn.
<: Hiermee geef je aan dat de waarde rechts groter moet zijn dan de waarde links.
<=: Hiermee geef je aan dat de waarde rechts groter dan of hetzelfde moet zijn als de waarde links.
<: Hiermee geef je aan dat de waarde links groter moet zijn dan de waarde rechts.
<=: Hiermee geef je aan dat de waarde links groter dan of hetzelfde moet zijn als de waarde rechts.

Je kan ook meerdere voorwaardes stellen. Daarvoor kan je de volgende gebruiken.
&&: De voorwaarde hiervoor en hierna moeten allebei kloppen.
||: De voorwaarde hiervoor of hierna moet kloppen. Het mag ook allebei kloppen. 


if (x == y)
if (x != y)
if (x < y)
if (x <= y)
if (x > y)
if (x >= y)
if (x == y && a == z)
if (x == y || a == z)
