Les 1: Je eigen website opzetten!


Stap 1: Open Visual Studio Code

Open de online app "Visual Studio Code" op je Chromebook. Hierin ga je alle code schrijven voor je website. Deze online app is te vinden via vscode.dev.

Stap 2: Maak een map aan voor je website

Ga naar "bestanden" op je Chromebook. Maak een nieuwe map aan om de bestanden voor je website in op te slaan. Je kan deze map noemen zoals je zelf wilt, maar de naam "Website" is het meest duidelijk.

Extra: Bestandsnamen

Voor het maken van een website zal je veel mapjes en bestanden moeten aanmaken. De namen van deze bestanden zijn bijna nooit van belang, maar het is handig om ze een duidelijke naam te geven. Als je dit niet doet, wordt alles namelijk erg onoverzichtelijk, en weet je niet welk bestand voor wat is.

Stap 3: Open map

Klik op "Open Folder" en kies de map die je zojuist hebt aangemaakt. Als het goed is, is deze map nog leeg. Hier moet je een paar keer toestemming geven om je map te openen in Visual Studio Code.

Stap 4: Maak een nieuwe webpagina aan.

Klik links op het icoontje met het bestandteken en een plusje. Hier kan je een nieuw bestand aanmaken. Dit wordt je eerste HTML bestand. Dit bestand bevat de code voor je website. Je kan hem zelf een naam geven, maar het is belangrijk dat je afsluit met het stukje ".html". Anders wordt het niet herkent als HTML bestand.

Extra: HTML

HTML staat voor "HyperText Markup Language". Dit is een programmeertaal voor het maken van webpagina's.

De landingspagina voor een website wordt vaak index.html genoemd. Dit komt omdat deze titel makkelijker te zoeken is in een zoekbalk. Je hoeft het stukje "index.html" namelijk nooit erachter te schrijven!

Stap 5: Open de webpagina

Ga naar je bestanden en zoek het html bestand dat je net hebt aangemaakt. Klik hier twee keer op om hem te openen. Als het goed is zie je nu een witte pagina.

Als je terug gaat naar je code, kan je hier iets intypen. Als je dit doet en je website ververst, kan je controleren of je de webpagina goed hebt opgezet!

Stap 6: HTML opzet

Kopieer het stukje code hiernaast en zet het in je html bestand. Dit stukje is het standaard begin stukje voor elk HTML bestand. Wat alles precies betekent is voor nu niet belangrijk. Als je extra tekst zou willen typen, dan kan dat tussen de "body" tags.


<!DOCTYPE html>
<html lang="en">
    <head>
        <title>Titel van je website</title>
        <meta charset="utf-8">
    </head>
    <body>
    </body>
</html>

Opdracht: Maak je website!

Volg de stappen zoals hierboven beschreven en zorg ervoor dat je website werkt! Probeer eens tekst toe te voegen tussen <body> en </body>. Krijg je die tekst te zien op je website?

Extra: Visual Studio Code

Visual Studio Code is een programma waarin je kunt programmeren. De online versie slaat automatisch je werk op, dus je hoeft niet bang te zijn dat je werk verloren gaat! Verder is het programma ontworpen om het coderen makkelijker te maken. Hier zal je vanzelf achter komen tijdens het maken van je website! Pas wel op dat je Visual Studio Code niet ververst, want dan moet je je map weer opnieuw koppelen.
