Variabelen toekennen en veranderen

Je kan variabelen op de volgende manieren toekennen en veranderen:
- =: Met = zeg je de waarde van een variabele. De nieuwe waarde vervangt de oude waarde.
- +=: Hiermee gebruik je de oude waarde, en tel je er een bepaald getal bij op.
- -=: Hiermee gebruik je de oude waarde, en haal je er een bepaald getal vanaf.
- *=: Hiermee gebruik je de oude waarde, en vermenigvuldig je het met een bepaald getal.
- /=: Hiermee gebruik je de oude waarde, en deel je het door een bepaald getal.
Dit komt aan bod in les 9. 

let nummer0 = 0;
nummer0 = 1;
nummer0 += 2;
nummer0 -= 2;
nummer0 *= 1;
nummer0 /= 1;

Operators

Om variabelen te vergelijken in een if statement, heb je een operator nodig. Deze zet je tussen 2 waardes in.
==: Hiermee geef je aan dat de waardes aan beide kanten precies hetzelfde moet zijn.
!=: Hiermee geef je aan dat de waardes aan beide kanten juist niet hetzelfde mogen zijn.
<: Hiermee geef je aan dat de waarde rechts groter moet zijn dan de waarde links.
<=: Hiermee geef je aan dat de waarde rechts groter dan of hetzelfde moet zijn als de waarde links.
<: Hiermee geef je aan dat de waarde links groter moet zijn dan de waarde rechts.
<=: Hiermee geef je aan dat de waarde links groter dan of hetzelfde moet zijn als de waarde rechts.
Dit komt aan bod in les 9. 

if (x == y)
if (x != y)
if (x < y)
if (x <= y)
if (x > y)
if (x >= y)

Extra voorwaarden

Soms wil je dat meerdere voorwaarden moeten kloppen bij een if statement, of dat 1 van de 2 voorwaarden moet kloppen. Daarvoor kan je de volgende gebruiken:
&&: De voorwaarde hiervoor en hierna moeten allebei kloppen.
||: De voorwaarde hiervoor of hierna moet kloppen. Het mag ook allebei kloppen.
Dit komt aan bod in les 9. 

if (x == y && a == z)
if (x == y || a == z)

while Loop

Een while loop loopt door zijn code heen zolang een bepaalde voorwaarde waar is. Eigenlijk werkt het hetzelfde als een if statement, maar dan loopt hij door zijn code heen.
Dit komt aan bod in les 10. 

while (nummer <= 10)
{
    CodeOmUitTeVoeren();
    nummer += 1;
}

for Loop

Een for loop heeft zelf een ingebouwde teller die je benoemt aan het begin van de loop. Tussen de haakjes komen 3 onderdelen te staan. De onderdelen worden gescheiden door een ;.
- Bij het eerste onderdeel wordt een nieuwe variabele als teller benoemd. In het voorbeeld is dat i.
- Bij het tweede onderdeel wordt aangegeven hoelang de loop door blijft gaan. In het voorbeeld is dat zolang de teller i onder de lengte van de array blijft.
- Bij het derde onderdeel wordt aangegeven wat er gebeurt na 1 loop. In het voorbeeld komt er 1 bij de teller i erbij. Op die manier gaat de loop dus niet oneindig door.
Dit komt aan bod in les 10. 

for (let i = 0; i < array.length; i++) 
{
    CodeOmUitTeVoeren();
}

forEach Loop

Een forEach loop kan alleen gebruikt worden bij een array. Deze loop loopt namelijk door alle elementen van een array. Je zal dus ook niet zo snel problemen krijgen omdat de computer oneindig vaak loopt.
Dit komt aan bod in les 10. 

array.forEach(element => 
{
    CodeOmUitTeVoeren();
});
