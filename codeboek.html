<html>

Hierin zitten alle tags voor je webspagina.
Deze tag komt aan bod in les 1. 

<html lang="en">
</html>

<head>

Hierin zit veel inhoudelijke informatie voor je website.
Deze tag komt aan bod in les 1. 

<head>
</head>

<title>

Hier kan je de titel van je webpagina aangeven. Deze komt vanboven te staan bij het tabblad. Dit zet je in de head van je webpagina.
Deze tag komt aan bod in les 1. 

<title>Titel van je webpagina</title>

<link>

Hiermee link je een CSS bestand. Gebruik hiervoor het juiste bestandspad. Dit zet je in de head van je webpagina.
Deze tag komt aan bod in les 4. 

<link rel="stylesheet" href="css/styles.css">

<body>

Hierin komt alle inhoud te staan van je webpagina. Zet dus alle inhoudstags (de tags hieronder) hiertussen!
Deze tag komt aan bod in les 1. 

<body>
</body>

<p>

Hiermee kan je normale tekst aangeven.
Deze tag komt aan bod in les 2. 

<p>Tekst</p>

<h4>Koptekst 4</h4>

<h5>

Hiermee kan je koptekst aangeven. Deze kop is kleiner dan <h4> en groter dan <h6>.

<h5>Koptekst 5</h5>

<header>

Met deze tag geef je de "header" aan van je webpagina. De header van een webpagina is het bovenste stukje van je webpagina, wat nog niet helemaal deel is van de inhoud. Zie het als de inleiding van een tekst.
Deze tag komt aan bod in les 2. 

<header>
</header>

<div>

Met deze tag geef je een "division" of verdeling aan van je webpagina. Met een div tag maak je de inhoud van je webpagina overzichtelijk. Je breekt namelijk de webpagina op in kleinere stukjes. Zie het als de alinea's van een tekst.
Deze tag komt aan bod in les 2. 

<div>
</div>

<footer>

Met deze tag geef je de "footer" aan van je webpagina. De footer van een webpagina is het onderste stukje van je webpagina, wat niet helemaal deel is van de inhoud. Zie het als het slot van een tekst.
Deze tag komt aan bod in les 2. 

<footer>
</footer>

<img>

Met deze tag zet je een afbeelding op je webpagina. Dit kan een online afbeelding zijn, of een afbeelding uit je eigen bestanden.
Deze tag komt aan bod in les 3. 

<img src="img/afbeelding.png"></img>
<img src="https://meesterthomas.github.io/img/MeesterThomas.png"></img>

<video>

Met deze tag zet je een video op je webpagina. Dit kan een Youtube video zijn, of een video uit je eigen bestanden.
Deze tag komt aan bod in les 3. 

<video src="video/filmpje.mp4" controls></video>

<audio>

Met deze tag zet je een video op je webpagina. Dit kan een Youtube video zijn, of een video uit je eigen bestanden.
Deze tag komt aan bod in les 3. 

<audio src="audio/geluid.mp3"></audio>
<audio src="https://audio12.forvo.com/audios/mp3/6/o/6o_9412304_118_3297390.mp3" controls></audio>

<p>Het volgende woord is <i>schuingedrukt</i>.</p>

<s>

Deze tag komt aan bod in les 3. 

<p>Het volgende woord is <s>doorgestreept</s>.</p>

<u>

Met deze tag kan je woorden onderstrepen.

<p>Het volgende woord is <u>onderstreept</u>.</p>

<mark>

Met deze tag kan je woorden markeren. De standaard kleur is geel, maar via CSS kan je met de "background-color" property de kleur aanpassen.
Deze tag komt aan bod in les 3. 

<p>Het volgende woord is <mark>gemarkeerd</mark>.</p>

Deze tag komt aan bod in les 7.
