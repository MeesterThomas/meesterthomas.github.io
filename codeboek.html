<header>

</header>

<div>

Met deze tag geef je een "division" of verdeling aan van je webpagina. Met een div tag maak je de inhoud van je webpagina overzichtelijk. Je breekt namelijk de webpagina op in kleinere stukjes. Zie het als de alinea's van een tekst.
Deze tag komt aan bod in les 2. 

<div>
</div>

<footer>

Met deze tag geef je de "footer" aan van je webpagina. De footer van een webpagina is het onderste stukje van je webpagina, wat niet helemaal deel is van de inhoud. Zie het als het slot van een tekst.
Deze tag komt aan bod in les 2. 

<footer>
</footer>

CSS


Tags opmaken

Via CSS kan je tags opmaken. Dit doe je vaak alleen als een tag over je hele website een bepaalde aanpassing heeft, zoals een lettertype.
Tags opmaken komt aan bod in les 4. 

HTML: <p>Dit is simpele tekst.</p>
CSS:
p
{
    background-color: blue;
}

Classes

Met classes kan je tags van een zelf gekozen categorie dezelfde opmaak geven. Een class is vaak breder dan een ID.
Classes komen aan bod in les 4. 

HTML: <p class="classNaam">Dit is simpele tekst.</p>
CSS:
.classNaam
{
    background-color: blue;
}

ID's

Met id's kan je tags van een zelf gekozen categorie dezelfde opmaak geven. Een ID is vaak specifieker dan een class.
ID's komen aan bod in les 4. 

HTML: <p id="classID">Dit is simpele tekst.</p>
CSS:
#classID
{
    background-color: blue;
}

background-color

Hiermee kan je de achtergrondkleur van een element aanpassen. Je kan een (Engelstalige) kleur noemen, of op het rgb spectrum een kleur kiezen (door op het gekleurde vakje te klikken in Visual Studio Code).
Deze property komt aan bod in les 4. 

background-color: blue;

color

Hiermee kan je de tekstkleur aanpassen. Je kan een (Engelstalige) kleur noemen, of op het rgb spectrum een kleur kiezen (door op het gekleurde vakje te klikken in Visual Studio Code).
Deze property komt aan bod in les 4. 

color: red;

font-size

Hiermee kan je de tekstgrootte aanpassen. Je kan de grootte aangeven in een maat van grootte, bijvoorbeeld px (pixels).
Deze property komt aan bod in les 4. 

font-size: 32px;

font-family

Hiermee kan je het lettertype aanpassen. Visual Studio Code geeft je de mogelijke lettertypes.
Deze property komt aan bod in les 4. 

font-family: system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 
'Open Sans', 'Helvetica Neue', sans-serif;

width

Hiermee kan je de breedte aanpassen. Je kan de grootte aangeven in een maat van grootte, bijvoorbeeld % (percentage).
Deze property komt aan bod in les 5. 

width: 50%;

height

Hiermee kan je de hoogte aanpassen. Je kan de grootte aangeven in een maat van grootte, bijvoorbeeld % (percentage).
Deze property komt aan bod in les 5. 

height: 10%;

border-width

Hiermee kan je de breedte van een border (rand) van een element aangeven. Je kan de grootte aangeven in een maat van grootte, bijvoorbeeld px (pixels).
Deze property komt aan bod in les 5. 

border-width: 5px;

border-style

Hiermee kan je de stijl van een border (rand) van een element aangeven. Visual Studio Code geeft je de mogelijke stijlen. "Solid" betekent dat de border een effen kleur heeft.
Deze property komt aan bod in les 5. 

border-style: solid;

border-color

Hiermee kan je de kleur van een border (rand) van een element aangeven. Je kan een (Engelstalige) kleur noemen, of op het rgb spectrum een kleur kiezen (door op het gekleurde vakje te klikken in Visual Studio Code).
Deze property komt aan bod in les 5. 

border-color: rgb(11, 11, 177);

border-radius

Hiermee kan je de ronding van de hoeken van een border (rand) van een element aangeven. Je kan de grootte aangeven in een maat van grootte, bijvoorbeeld px (pixels).
Deze property komt aan bod in les 5. 

border-radius: 5px;

padding

Hiermee kan je afstand van de content tot de border (rand) aanpassen. Het box model maakt de padding visueel. Je kan de grootte aangeven in een maat van grootte, bijvoorbeeld px (pixels). Als je 1 waarde geeft, is dat de waarde voor de afstanden aan alle 4 de kanten. Noem je er 2, is de eerste waarde voor de boven- en onderkant, en de tweede voor de linker- en rechterkant. Als je er 4 noemt, zijn alle waardes voor een aparte kant in de richting van de klok. Je kan de individuele waardes ook aanpassen.
Deze property komt aan bod in les 5. 

padding: 5px 2px 10px 8px;
padding-top: 5px;
padding-right: 2px;
padding-bottom: 10px;
padding-left: 8px;

margin

Hiermee kan je afstand vanaf de border (rand) tot een ander element aanpassen. Het box model maakt de margin visueel. Je kan de grootte aangeven in een maat van grootte, bijvoorbeeld px (pixels). Als je 1 waarde geeft, is dat de waarde voor de afstanden aan alle 4 de kanten. Noem je er 2, is de eerste waarde voor de boven- en onderkant, en de tweede voor de linker- en rechterkant. Als je er 4 noemt, zijn alle waardes voor een aparte kant in de richting van de klok. Je kan de individuele waardes ook aanpassen.
Deze property komt aan bod in les 5. 

margin: 5px 2px 10px 8px;
margin-top: 5px;
margin-right: 2px;
margin-bottom: 10px;
margin-left: 8px;

text-align

Hiermee kan je de positie van tekst aangeven. Je kan de positie aanpassen naar "left" (links), "right" (rechts) of "center" (midden).
Deze property komt aan bod in les 5. 

text-align: center;
