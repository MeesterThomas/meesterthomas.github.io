Les 1: Je eigen website opzetten!

In deze les leer je:
- Hoe je een nieuwe website aanmaakt.
- Hoe je je website kan bekijken.
- Hoe het begin van een HTML document eruitziet.

Les 2: Tekst, titels en kopjes

In deze les leer je:
- Hoe je tekst aan je website toevoegt.
- Wat het verschil is tussen header, footer en div.
- Wat het verschil is tussen p en h1 t/m h6.
- De HTML tags: p, br, h1, h2, h3, h4, h5, h6, header, footer en div.

Les 3: Belangrijke HTML tags

In deze les leer je:
- Hoe je afbeeldingen, videos en geluidsfragmenten aan je website toevoegt.
- Hoe je tekst verder kan opmaken.
- De HTML tags: img, video, audio, b, i, s, u en mark.

Les 4: CSS en opmaak

In deze les leer je:
- Wat een CSS bestand is en hoe je deze aan je website moet koppelen.
- Wat classes en id's zijn en hoe je ze kan gebruiken.
- Simpele voorbeelden van opmaak door CSS.
- De CSS properties: color, background-color, font-family en font-size.

Les 5: Het box model

In deze les leer je:
- Wat het box model is en waaruit het model bestaat.
- Hoe je je webpagina beter vorm kan geven.
- De CSS properties: padding, border-width, border-style, border-color, border-radius, margin en text-align.
